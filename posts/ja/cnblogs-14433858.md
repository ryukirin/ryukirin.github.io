## ファイル作成

### 空ファイルを作成する

```
os.mknod("test.txt")
```

**この方法は Windows では使えない。**

### ファイルを直接開き、存在しなければ作成する

```
import os

file = open("test01.txt",'w')
file.close()
with open("test01.txt", 'w', encoding="utf-8") as f:
    f.write("test")
with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
test
```

```
#封装的文件创建方法
def text_create(name):
    #此处是文件保存位置，默认为项目根目录
    savepath = ""
    filename = savepath + name + '.txt'
    filename = name + '.txt'
    file = open(filename, 'w')
    file.close()

text_create('test')
```

## ファイル書き込み

**ここでいう書き込みは、単なる `write` 操作だけを指しているわけではない。**

### write

`write` 操作は、ファイルの内容をすべて空にしてから改めて書き込む。

```
with open("test.txt", 'w', encoding="utf-8") as f:
    f.write("the 1st line\n")
#读取操作
with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
print("-------------------")
with open("test.txt", 'w', encoding="utf-8") as f:
    f.write("the 2nd line\n")

with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
the 1st line

-------------------
the 2nd line
```

​

### append

`append` 操作は、既存の内容の末尾に追記する。

```
with open("test.txt", 'a', encoding="utf-8") as f:
    f.write("the 3rd line\n")
#读取操作
with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
```

​

### 内容削除

```
with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
    
#f.truncate(n)从头开始，第n个字符后截断并清除
#即显示前n个字符
with open("test01.txt", 'a', encoding="utf-8") as f:
    f.truncate(2)

with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
    print("-------------")
    
#清除文件内容
with open("test01.txt", 'w', encoding="utf-8") as f:
    f.truncate()

with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
test
te
-------------
```

​

```
#删除文件
os.remove("test01.txt")

#检验文件是否存在
os.path.exists("test01.txt")
```

```
False
```

## ファイル読み込み

### open()+close()

```
#打开文件
f = open('test.txt','r',encoding = 'utf-8')

#再使用read()方法，查看文件里的内容：
print(f.read())

#关闭文件
f.close()
```

```
the 2nd line
the 3rd line
```

​

**注意**

- ファイルのエンコーディングを指定しない場合、ファイルのエンコーディングと現在のデフォルトエンコーディングが一致していないと、読み書きでエラーが発生する。Python 3 では `encoding` 引数でエンコーディングを指定できる。

- 最後には必ず `close()` を使ってファイルを閉じる。

- ファイルの読み書きそのものにエラーがある場合、`close()` を使っていてもファイルが正常に閉じられないことがある。

### with open()

`with` は `close()` メソッドを呼ぶのと同じ役割を持ち、ファイルを自動的に閉じてくれる。この方法の方が安全で、ファイルを閉じ忘れる問題も避けられる。

```
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
```

​

```
with open('test.txt','a',encoding = 'utf-8') as f:
    f.write("test\ntest2\ntest3\n")
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
test
test2
test3
```

​

### いくつかの読み込み方法

```
#读取全部
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
    print(f.tell()) #打印指针位置，read()操作之后指针在末尾
    f.seek(0) #移动指针（0-文件头，默认值；1-当前位置；2-文件尾）
    print(f.tell())
    print("-----------")

#读取第一行
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readline())
    print("-----------")
    
#读取前n个字符,这里n=2   
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readline(2))
    print("-----------")
    
#一次读取所有内容并按行返回list
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readlines())
    print("-----------")
```

```
the 2nd line
the 3rd line
test
test2
test3

48
0
-----------
the 2nd line

-----------
th
-----------
['the 2nd line\n', 'the 3rd line\n', 'test\n', 'test2\n', 'test3\n']
-----------
```
