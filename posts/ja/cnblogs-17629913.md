## 概要

この記事は Java を基礎から復習するシリーズの一部です。久しぶりに Java を触る前提で、文法、標準 API、オブジェクト指向、例外処理などを順番に確認しています。

## 学習メモ

Java は静的型付け言語なので、変数の型、メソッドの引数と戻り値、クラスの責務を明確に書く必要があります。最初は冗長に見えますが、型が明示されることで IDE の補完やコンパイル時チェックを活用しやすくなります。

オブジェクト指向では、クラスに状態と振る舞いをまとめ、必要に応じて継承やインターフェースで共通化します。ただし、継承を増やしすぎると構造が追いにくくなるため、まずは小さなクラスと明確なメソッドから始めるのが扱いやすいです。

## 実装時の注意

Java の学習では、コンパイルエラーを読む力が重要です。エラー行だけでなく、型の不一致、アクセス修飾子、例外処理の有無を確認すると原因を切り分けやすくなります。
