## 第五章 文字列

### 5.5 文字列をフォーマットする

- `format(String format, Object...args)`

`format`: フォーマット文字列。

- `args`: フォーマット文字列内のフォーマット指定子によって参照される引数。フォーマット指定子以外の引数があっても、それらの余分な引数は無視される。

`format(Local I, String format, Object...args)`

- `I`: フォーマット処理で適用するロケール。`null` の場合はローカライズしない。

- `format`: フォーマット文字列。

- `args`: フォーマット文字列内のフォーマット指定子によって参照される引数。フォーマット指定子以外の引数があっても、それらの余分な引数は無視される。

#### 日付と時刻文字列のフォーマット

出力される時刻文字列を、自分が欲しい形式に変える。

- 日付フォーマット

| 変換子 | 説明 | 例 |
| --- | --- | --- |
| %te | 月の中のある日(1~31) | 2 |
| %tb | 指定ロケールでの月の省略名 | Feb(en) 二月（中） |
| %tB | 指定ロケールでの月の正式名 | February(en) 二月（中） |
| %tA | 指定ロケールでの曜日の正式名 | Monday(en) 星期一（中） |
| %ta | 指定ロケールでの曜日の省略名 | Mon(en) 星期一（中） |
| %tc | すべての日付と時刻情報 | 星期二 三月 25 13:37:22 CST 2005 |
| %tY | 4 桁の年 | 2008 |
| %tj | 年内の何日目か(001~366) | 085 |
| %tm | 月 | 03 |
| %td | 月内の何日目か(1~31) | 02 |
| %ty | 2 桁の年 | 08 |

- 時刻フォーマット

| 変換子 | 説明 | 例 |
| --- | --- | --- |
| %tH | 2 桁の 24 時間制の時（00~23） | 14 |
| %tI | 2 桁の 12 時間制の時（01~12） | 05 |
| %tk | 2 桁の 24 時間制の時（0~23） | 5 |
| %tl | 2 桁の 12 時間制の時（1~12） | 10 |
| %tM | 2 桁の分（00~59） | 05 |
| %tS | 2 桁の秒（00~60） | 12 |
| %tL | 3 桁のミリ秒（000~999） | 920 |
| %tN | 9 桁のナノ秒 | 062000000 |
| %tp | 指定ロケールでの午前または午後 | 下午（中） pm(en) |
| %tz | GMT に対する RFC 82 形式の数値タイムゾーンオフセット | +0800 |
| %tZ | タイムゾーン省略形の文字列 | CST |
| %ts | 1970-01-01 00:00:00 から現在までの秒数 | 1206426646 |
| %tQ | 1970-01-01 00:00:00 から現在までのミリ秒数 | 1206426737453 |

| 変換子 | 説明 | 例 |
| --- | --- | --- |

#### 一般型フォーマット

任意の引数型に使える。

| 変換子 | 説明 | 例 |
| --- | --- | --- |
| %b %B | 結果を boolean 型としてフォーマットする | true |
| %h %H | 結果をハッシュコードとしてフォーマットする | A05A5198 |
| %s %S | 結果を文字列型としてフォーマットする | "abcd" |
| %c %C | 結果を文字型としてフォーマットする | 'a' |
| %d | 結果を 10 進整数としてフォーマットする | 40 |
| %o | 結果を 8 進整数としてフォーマットする | 11 |
| %x %X | 結果を 16 進整数としてフォーマットする | 4b1 |
| %e | 結果を科学的記数法の 10 進数としてフォーマットする | 1.700000e+01 |
| %a | 結果を有効桁数と指数を持つ 16 進浮動小数点数としてフォーマットする | 0X1.C000000000001P4 |
| %n | 結果はプラットフォーム固有の行区切り文字 |  |
| %% | 結果はリテラルの '%' | % |

```
import java.util.Date;

public class Java5_5 {
    public static void main(String[] args){
        // 日期格式化
        Date date = new Date();
        String year = String.format("%tY", date);
        String month = String.format("%tB", date);
        String day = String.format("%td", date);
        System.out.println(year + "年" + month + day + "日");

        // 时间格式化
        String hour = String.format("%tH", date);
        String minute = String.format("%tM", date);
        String second = String.format("%tS", date);
        System.out.println(hour + "时" + minute + "分" + second + "秒");

        // 格式化常见的日期时间组合
        String time = String.format("%tc", date);
        String from = String.format("%tF", date);
        System.out.println("全部的时间信息：" + time);
        System.out.println("年-月-日格式：" + from);

        // 常规类型格式化
        String str = String.format("%d", 400 / 2);
        String str2 = String.format("%b", 3 > 5);
        String str3 = String.format("%x", 200);
        System.out.println("400的一半是" + str);
        System.out.println("3>5正确吗？" + str2);
        System.out.println("200的16进制数是" + str3);
    }
}
```

### 5.6 正規表現を使う

正規表現は、文中のある文字が特定の形式を満たしているかどうかを判定するために使う。

正規表現で重要なのは、正規表現そのものを書けるようになること。

| メタ文字 | 正規表現での書き方 | 意味 |
| --- | --- | --- |
| . | . | 任意の 1 文字 |
| \d | \\d | 0~9 の任意の数字 |
| \D | \\D | 任意の非数字文字 |
| \s | \\s | 空白文字。例: '\t' '\n' |
| \S | \\S | 非空白文字 |
| \w | \\w | 識別子として使える文字。ただし "$" を含まない |
| \W | \\W | 識別子として使えない文字 |
| \p | \\p | 小文字 a~z |
| \p | \\p | 大文字 A~Z |
| \p | \\p | ASCII 文字 |
| \p | \\p | 英字 |
| \p | \\p | 10 進数字 0~9 |
| \p | \\p | 数字または英字 |
| \p | \\p | 句読点 `!"#$%'()*+,-./:;<=>?@[\\]^_`{\|}~` |
| \p | \\p | 可視文字 [\p{Alnum}\p{Punct}] |
| \p | \\p | 印字可能文字 [\p{Graph}\x20] |
| \p | \\p | スペースまたはタブ [\t] |
| \p | \\p | 制御文字 [\x00-\x1F\x7F] |

- 正規表現内で単純に `.` を表したい場合、エスケープ文字を付ける必要がある。

- `^` は先頭に一致し、`$` は末尾に一致する。

- 角括弧で複数の文字を囲むと、1 つのメタ文字を表せる。そのメタ文字は角括弧内の任意の 1 文字を表す。

例: `[abc]4` は `a4`、`b4`、`c4` の 3 つに一致する。

`[^456]` は 456 以外の任意の文字。

- `[a-r]` は a-r の任意の文字。

- `[a-zA-Z]` は任意の英字。

- `[a-e[g-z]]` は a-e または g-z の任意の文字。和集合。

- `[a-o&&[def]]` は d、e、f。積集合。

- `[a-d&&[^bc]]` は a、d。差集合。

| 限定修飾子 | 意味 | 例 |
| --- | --- | --- |
| ？ | 0 回または 1 回 | A? |
| * | 0 回以上 | A* |
| + | 1 回以上 | A+ |
| `{n}` | ちょうど n 回 | `A\{2}` |
| `{n,}` | 少なくとも n 回 | `A{3,}` |
| `{n,m}` | n~m 回 | `A{2,6}` |
| `^` | この正規表現に一致する文字列が先頭にある | `^A` |
| `ドル記号` | この正規表現に一致する文字列が末尾にある | `Aドル記号` |

*`$` は表の中でいつもフォーマットが崩れるので、ドル記号と書いた。*

```
public class Java5_6 {
    public static void main(String[] args){
        // \\w+: 一次或多次标识符 @ \\w+: 一次或多次标识符
        // (\.\w{2,3})*: .和两三个标识符 这一点主要是识别类似.com .cn .org等 这个组合出现0或多次
        // \.\w{2,3} 最后的.com .cn .org等
        // 这个正则表达式不一定可以匹配所有的邮件地址
        String regex = "\\w+@\\w+(\\.\\w{2,3})*\\.\\w{2,3}";
        String str1 = "aaa@";
        String str2 = "aaaaa";
        String str3 = "111@111ffyu.dfg.com";
        if (str1.matches(regex)){
            System.out.println(str1 + "是一个合法的E-mail地址");
        }
        else if (str2.matches(regex)){
            System.out.println(str2 + "是一个合法的E-mail地址");
        }
        else if (str3.matches(regex)){
            System.out.println(str3 + "是一个合法的E-mail地址");
        }
    }
}
```

### 5.7 文字列ビルダー

作成済みの文字列オブジェクトは長さが固定で、内容を変更・再コンパイルできない。

`+` は新しい文字や文字列を追加するために使えるが、新しい `String` インスタンスを生成し、メモリ内に新しい文字列オブジェクトを作る。

J2SE 5.0 で、可変の文字列シーケンスである `StringBuilder` クラスが追加された。

新しく作成された `StringBuilder` クラスの初期容量は 16 文字。超えた場合は自動的に長さが増える。

`StringBuilder` で文字列結果を出力したい場合は、`toString` メソッドを使える。

- `append()` 追記。`int` など任意の型のデータを受け取れる。

- `insert(int offset, arg)` 指定位置に挿入。任意の型のデータを受け取れる。

- `delete(int start, int end)` 指定位置の文字列を削除する。

```
public class Java5_7 {
    public static void main(String[] args) {
        String str = "";
        long starTime = System.currentTimeMillis();
        for (int i=0; i < 10000; i++){
            str = str + i;
        }
        long endTime = System.currentTimeMillis();
        long time = endTime - starTime;
        System.out.println("String消耗时间：" + time);
        StringBuilder builder = new StringBuilder("");
        starTime = System.currentTimeMillis();
        for (int j=0; j < 10000; j++){
            builder.append(j);
        }
        endTime = System.currentTimeMillis();
        time = endTime - starTime;
        // 耗时明显更短
        System.out.println("StringBuilder消耗时间：" + time);
    }
}
```
