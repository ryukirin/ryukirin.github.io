## __init__

`__init__` メソッドの第 1 引数は常に `self` で、作成されたクラスインスタンス自身を表す。

`__init__` メソッドがある場合、インスタンスを作成するときに空の引数では作成できず、`__init__` メソッドに対応する引数を渡す必要がある。ただし `self` は渡さなくてよい。Python インタプリタがインスタンス変数を自動的に渡してくれる。

```
class Student(object):
    def __init__(self, name, score):
        self.name = name
        self.score = score
student = Student("Hugh", 99)
print(student.name)
print(student.score)
```

```
Hugh
99
```

## メンバー関数

データのカプセル化。

`name` と `score` は `student` の内部変数なので、利用するときは関数でカプセル化できる。

```
class Student(object):
    def __init__(self, name, score):
        self.name = name
        self.score = score
    def Printinfo(self):
        print(self.name + ':' + str(self.score))

ryu = Student('ryu',98)
ryu.Printinfo()
```

```
ryu:98
```

ただしこの状態では、外部から `ryu` の `name` と `score` を変更できる。

```
ryu.name = 'tyo'
ryu.Printinfo()
```

```
tyo:98
```

`name` と `score` をプライベート変数にすれば、この問題を避けられる。この場合、`オブジェクト名.変数名` で具体的な値を取得することはできない。必要であれば getter / setter を追加して取得・変更する。

```
class Student(object):
    def __init__(self, name, score):
        self.__name = name #私有变量
        self.__score = score #私有变量
        
    def printInfo(self):
        print(self.__name + ':' + str(self.__score))
        
    def getName(self):
        print(self.__name)
        
    def getScore(self):
        print(self.__score)
        
    def setName(self, name):
        self.__name = name
        
    def setScore(self, score):
        self.__score = score
        
ryu = Student('ryu',98)
ryu.printInfo()

#print(ryu.__name) 这条会报错

ryu.name = 'tyo'
ryu.printInfo()
```

```
ryu:98
ryu:98
```

```
ryu.getName()
ryu.getScore()
ryu.setName('tyo')
ryu.setScore(100)
ryu.printInfo()
```

```
tyo
100
tyo:100
```

## 継承時

継承時に渡されるのは、そのとき実際に渡されたインスタンスであり、`self` を定義しているクラスのインスタンスとは限らない。

```
class Parent:
    def pprt(self):
        print(self)

class Child(Parent):
    def cprt(self):
        print(self)

c = Child()
c.cprt()
#等同于Child.pprt(c)，所以self指的依然是Child类的实例
#由于self中没有定义pprt()方法，所以沿着继承树往上找，发现在父类Parent中定义了pprt()方法，所以就会成功调用。
c.pprt()
p = Parent()
p.pprt()
```

```
<__main__.Child object at 0x000001EF6BBB2188>
<__main__.Child object at 0x000001EF6BBB2188>
<__main__.Parent object at 0x000001EF6BBB2F08>
```

## ディスクリプタクラス内

この場合、`self` が指すのはディスクリプタクラスのインスタンスである。

```
class Desc:
    
    #ins，用来表示调用它的类实例，若没有实例调用则为None
    def __get__(self, ins, cls):
        print('self in Desc: %s ' % self )
        print(self, ins, cls)
class Test:
    x = Desc()
    def prt(self):
        print('self in Test: %s' % self)
t = Test()
t.prt()
t.x
```

```
self in Test: <__main__.Test object at 0x0000025C9C4C5388>
self in Desc: <__main__.Desc object at 0x0000025C9C4C5308> 
<__main__.Desc object at 0x0000025C9C4C5308> <__main__.Test object at 0x0000025C9C4C5388> <class '__main__.Test'>
```

ここでの主な疑問は、「`Desc` クラスで定義した `self` は、それを呼び出したインスタンス `t` になるべきではないのか。なぜ `Desc` クラスのインスタンスになっているのか」という点だと思う。

ここで呼んでいるのは `t.x` である。つまり、`Test` クラスのインスタンス `t` の属性 `x` を参照している。インスタンス `t` 自体には属性 `x` が定義されていないため、クラス属性 `x` を探しにいく。そしてその属性はディスクリプタ属性であり、`Desc` クラスのインスタンスである。そのため、ここでは `Test` のどのメソッドも呼んでいない。

クラスから直接属性 `x` を呼び出しても、同じ結果が得られる。

下は `t.x` を `Test.x` に変更して実行した結果。

```
class Desc:
    def __get__(self, ins, cls):
        print('self in Desc: %s ' % self )
        print(self, ins, cls)
class Test:
    x = Desc()
    def prt(self):
        print('self in Test: %s' % self)
t = Test()
t.prt()
Test.x
```

```
self in Test: <__main__.Test object at 0x000001EF6BF34408>
self in Desc: <__main__.Desc object at 0x000001EF6B8FCF88> 
<__main__.Desc object at 0x000001EF6B8FCF88> None <class '__main__.Test'>
```

## self を使わない場合

- クラスメソッド: インスタンスから呼び出すというより、クラスから呼び出すことを重視するメソッド。たとえばコンストラクタのオーバーロードのように、データ変換に使える。メソッドの第 1 引数は class オブジェクトになる。

- 静的メソッド: 主にそのクラスと論理的に関連する処理を扱う。たとえばデータ検証など。第 1 引数は class オブジェクトでもインスタンス `self` でもなくてよい。

```
class Student(object):
    def __init__(self, name, score):
        self.name = name
        self.score = score
        
    def Printinfo(self):
        print(self.name + ':' + str(self.score))
    
    @classmethod
    def classFunction(cls, info):
        name, score = info.split(' ')
        stu = cls(name, score)
        return stu
    
    @staticmethod
    def ifgood(score):
        return score>=90

ryu = Student('ryu',98)
ryu.Printinfo()
#ryu.classFunction() 会报错classFunction() takes 0 positional arguments but 1 was given
li = Student.classFunction('li 89')
li.Printinfo()
li.ifgood(int(li.score))
Student.ifgood(99)
```

```
ryu:98
li:89

True
```
