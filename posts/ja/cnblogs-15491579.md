# 吃瓜教程 | Datawhale 10月（5）

## 支持ベクトルマシン

幾何学的な観点では、線形分離可能なモデルにおける支持ベクトルマシンは、正例と負例のどちらからも最も遠い超平面を探す手法である。

パーセプトロンと比べると、解が一意で、偏りが少なく、汎化性能もよい。

### 超平面

\\(n\\) 次元空間における超平面 \\(\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b=0\\right. , 其中  \\boldsymbol{w}, \\boldsymbol{x} \\in \\mathbb{R}^{n}  )\\) ：

- 超平面の方程式は一意ではない。

- 法線ベクトル \\(\\boldsymbol{w}\\) と変位項 \\(b\\) によって、一つの超平面が一意に決まる。

- 法線ベクトル \\(\\boldsymbol{w}\\) は超平面に垂直である。\\(\\boldsymbol{w}, b\\) をスケーリングするとき、倍率が負なら法線ベクトルの向きが変わる。

- 法線ベクトル \\(\\boldsymbol{w}\\) が指す半空間を正の空間、反対側を負の空間とする。

- 任意の点 \\(\\boldsymbol{x}\\) から超平面までの距離は次の式で表される。

\\[r=\\frac{\\left|\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b\\right|}{\\|\\boldsymbol{w}\\|}
\\]

### 幾何マージン

与えられたデータセット \\(X\\) と超平面 \\(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b=0\\) に対して、データセット \\(X\\) 内の任意のサンプル点 \\(\\left(\\boldsymbol{x}_{i}, y_{i}\\right), y_{i} \\in\\{-1,1\\}, i=1,2, \\ldots, m\\) の、その超平面に対する幾何マージンを次のように定義する。

\\[\\gamma_{i}=\\frac{y_{i}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{i}+b\\right)}{\\|\\boldsymbol{w}\\|}
\\]

正しく分類されている場合は \\(\\gamma_{i}>0\\) であり、このとき幾何マージンは点から超平面までの距離と等価になる。正しく分類されていない場合は \\(\\gamma_{i}<0\\) である。
与えられたデータセット \\(X\\) と超平面 \\(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b=0\\) に対して、データセット \\(X\\) の超平面に対する幾何マージンを、データセット \\(X\\) 内の全サンプル点の幾何マージンの最小値として定義する。

\\[\\gamma=\\min _{i=1,2, \\ldots, m} \\gamma_{i}
\\]

### 支持ベクトルマシン

目的は、データセットに対する幾何マージンが最大になる超平面を求め、その上で `sign` 関数を使って分類を実現することである。

\\[y=\\operatorname{sign}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b\\right)=\\left\\{\\begin{aligned}
1, & \\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b>0 \\\\
-1, & \\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}+b<0
\\end{aligned}\\right.
\\]

したがって本質的にはパーセプトロンと同じく、超平面を求める問題である。

幾何マージンが最大となる超平面は、正負サンプルのどちらからも最も遠い超平面である。その理由は次の二点。

- 超平面が正負サンプルを正しく分けられていない場合、幾何マージン最小の点は誤分類点であるため、\\(\\gamma<0\\) になる。

- 超平面が正しく分けられている場合、\\(\\gamma \\geqslant 0\\) であり、中央に近いほど \\(\\gamma\\) は大きくなる。

戦略: 線形分離可能なデータセット \\(X\\) が与えられ、\\(X\\) の中で幾何マージンが最小のサンプルを \\(\\left(\\boldsymbol{x}_{\\min }, y_{\\min }\\right)\\) とする。このとき、支持ベクトルマシンが超平面を探す過程は、次の制約付き最適化問題へ変換できる。

\\[\\begin{array}{ll}
\\max \\quad \\gamma \\\\
\\text { s.t. } \\quad \\gamma_{i} \\geqslant \\gamma, \\quad i=1,2, \\ldots, m \\\\
\\end{array}
\\]

\\[\\begin{array}{ll}
\\max _{\\boldsymbol{w}, b} \\frac{y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right)}{\\|\\boldsymbol{w}\\|} \\\\
\\text { s.t. } \\quad \\frac{y_{i}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{i}+b\\right)}{\\|\\boldsymbol{w}\\|} \\geqslant \\frac{y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right)}{\\|\\boldsymbol{w}\\|},  \\quad i=1,2, \\ldots, m \\\\
\\end{array}
\\]

\\[\\begin{array}{ll}
\\max _{\\boldsymbol{w}, b} \\frac{y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right)}{\\|\\boldsymbol{w}\\|} \\\\
\\text { s.t. } \\quad y_{i}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{i}+b\\right) \\geqslant y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right), \\quad i=1,2, \\ldots, m
\\end{array}
\\]

この問題の最適解を \\(\\left(\\boldsymbol{w}^{*}, b^{*}\\right)\\) と仮定すると、\\(\\left(\\alpha \\boldsymbol{w}^{*}, \\alpha b^{*}\\right), \\alpha \\in \\mathbb{R}^{+}\\) も最適解であり、超平面も変わらない。そのため、上の最適化問題に解ける一意解を持たせるには、\\(\\boldsymbol{w}, b\\) に一定の制限を加える必要がある。そこで \\(y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right)=1\\) としてよい。特定の \\(\\left(\\boldsymbol{x}_{\\min }, y_{\\min }\\right)\\) に対して、
\\(y_{\\min }\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{\\min }+b\\right)=1\\) を満たす \\(\\alpha\\) はただ一つだからである。したがって、上の最適化問題はさらに次の形へ変換される。

\\[\\begin{array}{ll}
\\max _{\\boldsymbol{w}, b} & \\frac{1}{\\|\\boldsymbol{w}\\|} \\\\
\\text { s.t. } & y_{i}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{i}+b\\right) \\geqslant 1, \\quad i=1,2, \\ldots, m
\\end{array}
\\]

後続計算を扱いやすくするため、さらに等価変形を行う。計算の都合で二乗と `1/2` を加える。

\\[\\begin{array}{ll}
\\min _{\\boldsymbol{w}, b} & \\frac{1}{2}\\|\\boldsymbol{w}\\|^{2} \\\\
\\text { s.t. } & 1-y_{i}\\left(\\boldsymbol{w}^{\\mathrm{T}} \\boldsymbol{x}_{i}+b\\right) \\leqslant 0, \\quad i=1,2, \\ldots, m
\\end{array}
\\]

この最適化問題は不等式制約付きの最適化問題であり、凸最適化問題でもある。そのため、凸最適化問題を解くための専用手法をそのまま適用できる。支持ベクトルマシンでは通常、Lagrange 双対を使ってこの問題を解く。
