## 第六章 配列

### 6.1 配列の概要

配列は、同じデータ型を持つデータの集合である。

### 6.2 一次元配列の作成と使用

本質的には、同じ型のデータを線形に並べた集合である。

#### 一次元配列を作成する

配列の添字は 0 から始まる。

- 先に宣言し、その後 `new` 演算子でメモリを割り当てる。

`配列要素型 配列名[]`

- `配列要素型[] 配列名`

宣言直後にはまだアクセスできない。メモリ空間を割り当てる必要があり、長さを明示しなければならない。

`配列名 = new 配列要素型[配列要素の数]`

宣言と同時に配列へメモリを割り当てる。

- `配列要素型 配列名[] = new 配列要素型[配列要素の数]`

- `配列要素型[] 配列名 = new 配列要素型[配列要素の数]`

#### 一次元配列を初期化する

- 配列にメモリを割り当てることも初期化の一種。この時点ではすべての要素がデフォルトで 0 になる。

- `new` の中で初期化する。例: `int arr[] = new int[]{1, 2, 3, 4};`

- `new` を省略して直接初期化する。例: `int arr[] = {1, 2, 3, 4};`

配列の長さは一度定義すると変更できない。

#### 一次元配列を使用する

`for` ループで走査する。

```
public class Java6_1_2 {
    public static void main(String[] args) {
        //创建一维数组
        //先声明 再用new运算符进行内存分配
        int arr[];
        int[] arr1;
        arr = new int[3];
        arr1 = new int[4];
        // 声明的同时为数组分配内存
        int arr2[] = new int[5];

        // 初始化一维数组
        int[] arr3 = new int[6];
        System.out.println(arr3[0]);
        int arr4[] = {1, 2, 3, 4};

        // 使用一维数组
        for (int i = 0; i<arr4.length; i++){
            System.out.println(arr4[i]);
        }
        for (int i : arr4){
            System.out.println(i);
        }
    }
}
```

### 6.3 二次元配列の作成と使用

#### 二次元配列の作成

- 先に宣言し、その後 `new` 演算子でメモリを割り当てる。

`配列要素型 配列名[][]`

- `配列要素型[][] 配列名`

高次元配列にメモリ空間を割り当てる。

各次元に直接メモリを割り当てる。例: `a = new int[2][4]`

- 各次元に個別にメモリを割り当てる。例: `a = new int[2][]`、`a[0] = new int[2]`、`a[1] = new int[3]`

宣言と同時に配列へメモリを割り当てる。

#### 二次元配列の初期化

例: `int arr[][] = {{1, 2, 3}, {4, 5, 6}}`

#### 二次元配列を使用する

```
public class Java6_3 {
    public static void main(String[] args) {
        int arr[][] = {{1, 2, 3}, {4, 5}};
        for (int i=0; i<arr.length; i++){
            for (int j=0; j<arr[i].length; j++){
                System.out.println(arr[i][j]);
            }
        }
        for (int[] i : arr){
            for (int j : i){
                System.out.println(j);
            }
        }
    }
}
```

### 6.4 配列の基本操作

`Arrays` クラスを使う。

`import java.util.Arrays;`

#### 配列を走査する

前の 2 節で 2 つの方法を書いた。

- `for`

- `foreach`

#### 配列要素を埋める・置換する

- `Arrays.fill(int[] a, int value)`

`a`: 要素を置換する配列。

- `value`: 配列に格納する値。

`Arrays.fill(int[] a, int fromIndex, int toIndex, int value)`

- `a`: 要素を置換する配列。

- `fromIndex`: 指定値で埋める最初の要素のインデックス（含む）。

- `toIndex`: 指定値で埋める最後の要素のインデックス（含まない）。

- `value`: 配列に格納する値。

#### 配列をソートする

`Arrays.sort()` は昇順ソート。

`String` 型の場合は辞書順の昇順になる。

#### 配列をコピーする

- `Arrays.copyOf(arr, int newLength)`

- `Arrays.copyOfRange(arr, int formIndex, int toIndex)`

#### 配列検索

`Arrays.binarySearch` は二分探索で検索する。

使う前に必ずソートしておく必要がある。

- `Arrays.binarySearch(Object[] a, Object key)`

`a`: 検索対象の配列。

- `key`: 検索する値。

- 戻り値はインデックス。

`Arrays.binarySearch(Object[] a, int formIndex, int toIndex, Object key)`

```
import java.util.Arrays;
public class Java6_4 {
    public static void main(String[] args) {
        // 填充替换数组元素
        int arr[] = {1, 2, 3, 4, 5};
        Arrays.fill(arr, 5);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");
        Arrays.fill(arr, 2, 3, 1);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");

        // 对数组进行排序
        Arrays.sort(arr);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");

        // 复制数组
        int newArr[] = Arrays.copyOf(arr, 10);
        for (int i : newArr) {
            System.out.print(i);
        }
        System.out.println("");

        int newArr1[] = Arrays.copyOfRange(arr, 0, 3);
        for (int i : newArr1) {
            System.out.print(i);
        }
        System.out.println("");

        // 数组搜索
        System.out.println("1的索引为" + Arrays.binarySearch(arr, 1));
        // 结果是2 因为会返回第一找到的key的索引 binarySearch是二分法 故第一个找的位置是正中间
        System.out.println("1的索引为" + Arrays.binarySearch(arr, 5));
    }
}
```
