# How to create a custom Memory class 自作 Memory クラスを作る

この記事では、LangChain で使う Memory クラスを自作する。

原文: [How to create a custom Memory class](https://python.langchain.com/docs/modules/memory/how_to/custom_memory)

## 翻訳

LangChain には事前定義された memory type がいくつかあるが、自分のアプリケーションに合わせて独自の memory type を追加したくなる場面は十分にある。このノートでは、その追加方法を紹介する。

ここでは `ConversationChain` に custom memory type を追加する。独自 memory class を追加するには、基底となる memory class を import し、それを継承したクラスを作る。

```
from langchain import OpenAI, ConversationChain
from langchain.schema import BaseMemory
from pydantic import BaseModel
from typing import List, Dict, Any
```

この例では、spaCy を使って entity を抽出し、entity 情報を単純なハッシュテーブルに保存する custom memory class を書く。会話中は input のテキストを見て、すべての entity を抽出し、それらに関する既知情報を context に入れる。

- 注意: この実装は非常に単純で壊れやすく、本番環境では役に立たない可能性がある。目的は、custom memory を追加できることを示すことにある。

このために spaCy が必要になる。

### spaCy 簡単な紹介

spaCy はすぐに使い始められる NLP 開発ライブラリである。ここでは、まず言語モデルを読み込み、文を入力して処理する。処理中にトークン化、品詞タグ付けなど複数の NLP タスクが同時に行われ、結果は `doc` クラスに入る、という理解でよい。

クイックスタート: [使用spaCy做进阶自然语言处理](https://course.spacy.io/zh)

### custom memory の例

```
import spacy

nlp = spacy.load("en_core_web_lg")
```

```
class SpacyEntityMemory(BaseMemory, BaseModel):
    """entity 情報を保存する memory class"""

    # entity 情報を保存する辞書を定義する
    entities: dict = {}
    # prompt に追加する entity 情報を絞り込む key
    memory_key: str = "entities"

    def clear(self):
        self.entities = {}

    @property
    def memory_variables(self) -> List[str]:
        """prompt に追加する変数を定義する"""
        return [self.memory_key]

    def load_memory_variables(self, inputs: Dict[str, Any]) -> Dict[str, str]:
        """memory variables を読み込む。この例では entities"""
        # input テキストを取得して spaCy に通す
        doc = nlp(inputs[list(inputs.keys())[0]])
        # entity に関する既知情報を抽出する。存在する場合のみ
        entities = [
            self.entities[str(ent)] for ent in doc.ents if str(ent) in self.entities
        ]
        # entity の統合情報を返し、context に入れる
        return {self.memory_key: "\n".join(entities)}

    def save_context(self, inputs: Dict[str, Any], outputs: Dict[str, str]) -> None:
        """今回の会話 context を buffer に保存する"""
        # input テキストを取得して spaCy に通す
        text = inputs[list(inputs.keys())[0]]
        doc = nlp(text)
        # 言及された各 entity について、情報を辞書へ保存する
        for ent in doc.ents:
            ent_str = str(ent)
            if ent_str in self.entities:
                self.entities[ent_str] += f"\n{text}"
            else:
                self.entities[ent_str] = text
```

次に、entity 情報とユーザー input を受け取る prompt を定義する。

```
from langchain.prompts.prompt import PromptTemplate

template = """The following is a friendly conversation between a human and an AI. The AI is talkative and provides lots of specific details from its context. If the AI does not know the answer to a question, it truthfully says it does not know. You are provided with information about entities the Human mentions, if relevant.

Relevant entity information:
{entities}

Conversation:
Human: {input}
AI:"""
prompt = PromptTemplate(input_variables=["entities", "input"], template=template)
```

そして、これらをまとめる。

```
llm = OpenAI(temperature=0)
conversation = ConversationChain(
    llm=llm, prompt=prompt, verbose=True, memory=SpacyEntityMemory()
)
```

## 個人テスト例

ここでは、GPT-3.5 が毎回生成する JSON の特定 key の値を保存できる custom memory が欲しかった。

この例での JSON 形式は次のとおり。

```
{
  "current_step": 現在のステップ名;
  "result_info":
	{
    	"step": 既定ステップ内にいくつステップがあれば、step リストにもその数だけ json を生成する
    	[{
      		"name":ステップ名。既定ステップ内で生成されるステップ名;
      		"ok": ユーザーへの確認が完了したか、はい/いいえ;
		}, ...];
    "is_over": すべてユーザーに確認済みか、はい/いいえ;
    "next_step": "is_over" が "はい" の場合は次のステップ名を書く。"is_over" が "いいえ" の場合は "";
	};
  "reply": 会議ファシリテーターの発話。必ず内容がある;
}
```

取得したい key は `"reply"` なので、新しく `RecentKConversationMemory` クラスを作る。

*注意: この `RecentKConversationMemory` クラスは汎用性がかなり低く、このタスクでのみ使える。*

```
from langchain.schema import BaseMemory
from typing import List, Dict, Any


class RecentKConversationMemory(BaseMemory):
    """直近 5 ラウンドの会話から、JSON ファイル内の特定 key の値を保存する memory class"""
    # 直近会話を保存するリストを定義する。既定は空
    recent_conversations: list = []
    # prompt に追加する entity 情報を絞り込む key。既定は "recent_conversations"
    memory_key: str = "recent_conversations"
    # prefix を定義する。既定は "Human" と "AI"
    human_prefix: str = "Human"
    ai_prefix: str = "AI"
    # 保持するラウンド数。既定は 5
    k: int = 5

    def clear(self):
        self.recent_conversations = []

    @property
    def memory_variables(self) -> List[str]:
        """prompt に追加する変数を定義する"""
        return [self.memory_key]

    def load_memory_variables(self, inputs: Dict[str, Any]) -> Dict[str, str]:
        """memory variables を読み込む。ここでは特定 key の内容"""
        recent_json_value = ""

        # 直近 k ラウンドの会話を走査する
        for conversation in self.recent_conversations[-self.k:]:
            # key が "reply" の内容が存在するか確認する
            if "reply" in eval(conversation["output"]["response"]).keys():
                human = f"{self.human_prefix}: " + conversation["input"]['input']
                ai = f"{self.ai_prefix}: " + eval(conversation["output"]["response"])["reply"]
                recent_json_value += "\n" + "\n".join([human, ai])

        return {self.memory_key: recent_json_value}

    def save_context(self, inputs: Dict[str, Any], outputs: Dict[str, Any]) -> None:
        """今回の会話 context を buffer に保存する"""
        # 各ラウンドの入力と出力を一つの会話として保存する
        conversation = {
            "input": inputs,
            "output": outputs
        }
		# 5 個を超えたら、最初の conversation を pop する
        self.recent_conversations.append(conversation)
        if len(self.recent_conversations) > self.k:
            self.recent_conversations.pop(0)
```
