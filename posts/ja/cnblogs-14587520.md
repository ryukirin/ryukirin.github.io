### パラメータ

- input_size: 入力の次元。`embedding_size` に相当する。

- hidden_size: `h` の次元。

- num_layers: 積み重ねる LSTM 層の数。デフォルトは 1。

- bias: 隠れ状態に bias を付けるかどうか。デフォルトは true。bias は偏置値、またはオフセット値である。bias がなければ、0 を中心軸、または 0 を起点とすることになる。

- batch_first: True に設定すると、入力データの次元のうち最初の次元が batch になる。デフォルトは False。詳細は [Pytorch的参数“batch_first”的理解](https://www.jianshu.com/p/41c15d301542) を参照。

- dropout: デフォルトは 0。最後の RNN 層以外の RNN 層の後ろに dropout 層を追加するかどうかを指定する。入力値は 0 から 1 の小数で、確率を表す。0 は dropout の確率が 0、つまり dropout しないことを意味する。

- bidirectional: 双方向伝播にするかどうか。デフォルトは False。

#### dropout

dropout は名前のとおり「取り除く」という意味である。ニューラルネットワークの一部のニューロンを取り除くので、dropout 層と呼ばれる。目的は過学習の防止である。

最初の batch を訓練するときは、以下のような流れになる。

- 各ニューラルネットワーク層に対して dropout の確率を設定する。

- 対応する確率に従って一部のニューロンを取り除き、残ったニューロンと重みパラメータを訓練して更新し、その値を保持する。

- すべてのパラメータを更新したあと、再び同じ確率に従って一部のニューロンを取り除き、訓練を始める。新しく訓練に使われるニューロンが 1 回目でも訓練されていた場合は、そのパラメータを引き続き更新する。2 回目で取り除かれたが、1 回目ではすでに更新済みだったニューロンについては、その重みを保持して変更しない。第 n 回の batch で dropout されなかったときに、また更新される。

```
#num_layers

import torch.nn as nn
import torch
x = torch.rand(10,24,100)

#torch.nn.lstm(input_size=100,hidden_size=16,num_layers=2)
#输入维度100，h维度16，lstm层数2
#以训练句子为例子，假如每个词是100维的向量，每个句子含有24个单词，一次训练10个句子。
#那么batch_size=10,seq=24,input_size=100。(seq指的是句子的长度，input_size作为一个x_t的输入) 
lstm = nn.LSTM(100,16,num_layers=2)
output,(h,c) = lstm(x)
print(output.size())
print(h.size())
print(c.size())
```

torch.Size([10, 24, 16])
torch.Size([2, 24, 16])
torch.Size([2, 24, 16])

```
#num_layers,bidirectional

import torch.nn as nn
import torch
x = torch.rand(10,24,100)
lstm = nn.LSTM(100,16,bidirectional=True)
output,(h,c) = lstm(x)
print(output.size())
print(h.size())
print(c.size())
```

torch.Size([10, 24, 32])
torch.Size([2, 24, 16])
torch.Size([2, 24, 16])

```
import torch.nn as nn
import torch
x = torch.rand(10,24,100) #batch,seq,input_size
h0 = torch.rand(1,24,16)# num_layers*num_directions, batch, hidden_size
c0 = torch.rand(1,24,16)
lstm = nn.LSTM(100,16)
output,(h,c) = lstm(x,(h0,c0))
print(output.size())
print(h.size())
print(c.size())
```

torch.Size([10, 24, 16])
torch.Size([1, 24, 16])
torch.Size([1, 24, 16])

最後にパラメータの次元を付けておく。

説明すると、1 行目は LSTM を構築するときの入力次元で、後続の数行は LSTM が使うパラメータの次元を入力次元から推定したものである。

双方向の場合は `num_directions=2`、そうでなければ `num_directions=1`。

| パラメータ | 次元1 | 次元2 | 次元3 |
| --- | --- | --- | --- |
| lstm=nn.LSTM | input_size | hidden_size | num_layers |
| x | seq_len | batch | input_size |
| h0 | num_layers×num_directions | batch | hidden_size |
| c0 | num_layers×num_directions | batch | hidden_size |
| output | seq_len | batch | num_directions×hidden_size |
| hn | num_layers×num_directions | batch | hidden_size |
| cn | num_layers×num_directions | batch | hidden_size |
