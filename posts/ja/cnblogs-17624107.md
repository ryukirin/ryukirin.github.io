## 第五章 文字列

### 5.3 文字列情報の取得

#### 長さ情報

`str.length();`

#### 文字列検索

インデックス範囲は `(0, length-1)`。

- `indexOf()`

最初に出現したインデックス位置を返す。存在しない場合は -1 を返す。

- `lastIndexOf()`

最後に出現したインデックス位置を返す。存在しない場合は -1 を返す。

検索対象が `""` の場合、返るのは長さである。

#### 指定インデックス位置の文字を取得する

`str.charAt(int index)`

```
public class Java5_3 {
    public static void main(String[] args){
        String s = "hello world!";
        int len = s.length();
        System.out.println(len);

        // indexOf()
        int idx1;
        idx1 = s.indexOf('l');
        System.out.println(idx1);

        // lastIndexOf()
        int idx2, idx3;
        idx2 = s.lastIndexOf('l');
        idx3 = s.lastIndexOf("");
        System.out.println(idx2);
        System.out.println(idx3);
        System.out.println(s.length());

        // 获取指定索引位置的字符
        System.out.println(s.charAt(4));
    }
}
```

### 5.4 文字列操作

#### 文字列を取得する

文字列を切り出す。

- `substring(int beingIndex)`

開始位置から末尾まで切り出す。

- `substring(int beingIndex, int endIndex)`

開始位置と終了位置を指定する。

#### 空白を取り除く

`trim()`

前後の空白だけを無視する。

#### 文字列置換

`str.replace(char oldChar, char newChar)`

複数回出現している場合はすべて置換される。

#### 文字列の開始と終了を判定する

- `startsWith()` ある文字列で始まるか判定する。

- `endsWith()` ある文字列で終わるか判定する。

#### 文字列が等しいか判定する

`==` は 2 つの変数のアドレスが同じかどうかを比較する。

しかし 2 つの文字列は、内容が同じでもアドレスが違う場合がある。そのため結果は false になり得る。

したがって `equals()` と `equalsIgnoreCase()` を使う。

後者は大文字小文字を無視する。

#### 辞書順で 2 つの文字列を比較する

Unicode 値に基づく。比較対象より前なら負数、後なら正数、同じなら 0。

`str.compareTo()`

#### 文字の大文字・小文字変換

数字と非文字は影響を受けない。

- `toLowerCase()` 小文字に変換する。

- `toUpperCase()` 大文字に変換する。

#### 文字列分割

- `spilt(String sign)` 指定文字で分割する。

- `spilt(String sign, int limit)` 指定文字で分割し、回数を制限する。

```
public class Java5_4 {
    public static void main(String[] args){
        String s = " maybe this is an example ";

        // 获取字符串
        String s1 = s.substring(2);
        System.out.println(s1);

        String s2 = s.substring(2, 6);
        System.out.println(s2);

        // 去除空格
        String s3 = s.trim();
        System.out.println(s3);

        // 字符串替换
        String s4 = s.replace("a", "b");
        System.out.println(s4);

        // 判断字符串的开始和结尾
        System.out.println(s.startsWith(" "));
        System.out.println(s.endsWith("z"));

        // 判断字符串是否相等
        String s5 = " maybe this is an example ";
        String s6 = " Maybe this is an EXAMPLE ";
        System.out.println(s.equals(s5));
        System.out.println(s.equalsIgnoreCase(s6));

        // 按照字典顺序比较两个字符串
        System.out.println(s.compareTo(s6));
        System.out.println(s.compareTo(s5));

        // 字母大小写转换
        System.out.println(s.toUpperCase());

        // 字符串分割
        String[] ss = s.split(" ");
        String[] ss1 = s.split(" ", 3);
        // 遍历数组
        for (String i:ss) {
            System.out.println(i);
        }
        for (String i:ss1) {
            System.out.println(i);
        }
    }
}
```
