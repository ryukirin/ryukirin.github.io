以前、アイコンを削除して Steam ライブラリからデスクトップショートカットを追加し直す、というような方法をいろいろ試したが、全部だめだった。

それどころか、一度アンインストールして再インストールまでした。

Steam ゲームのアイコンを保存している専用フォルダも見に行ったが、変更したいゲームのアイコンはそもそも存在しなかった。

その後、アイコンは単純に直接変更すればよいことに気づいた。

ここでは Octopath Traveler を例にする。壊れていたのがこのゲームで、今は直っている。

### デスクトップショートカットを右クリック -> プロパティ -> Web ドキュメント -> アイコンの変更

![](/assets/posts/cnblogs-14529720/image-01.png)

以前見た手順どおりなら、「参照」をクリックして `インストール先\Steam\steam\games` を開くと `.ico` のアイコンファイルが見えるはずだが、私の環境では見つからなかった。

そこで、Octopath Traveler の `.exe` ファイルがあるディレクトリを直接探した。

### Steam ライブラリ -> Octopath Traveler -> 設定 -> 管理 -> ローカルファイルを閲覧

![](/assets/posts/cnblogs-14529720/image-02.png)

通常は `インストール先\Steam\steamapps\common\ゲーム名` の中にあるはず。

Octopath Traveler はゲームのルートディレクトリで直接見つかった。隣の Don't Starve は `bin` ディレクトリの中にあった。
