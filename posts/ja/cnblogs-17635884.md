## 第六章 配列

### 6.5 配列ソートアルゴリズム

#### バブルソート

最もよく使われる配列ソートアルゴリズムの 1 つ。常に小さい数を前に、大きい数を後ろに移動する。水中の泡のように見えるため、この名前が付いている。

- 基本思想

隣接する要素の値を比較し、条件を満たせば要素値を交換する。

小さい要素を配列の前へ、大きい要素を配列の後ろへ移動する。

図を見るとかなり分かりやすい。

![](/assets/posts/cnblogs-17635884/image-01.gif)

図の出典: 菜鸟教程

第 1 ラウンドが終わると、最大値はすでに最後に沈んでいる。第 2 ラウンドでは、最後から 2 番目まで比較すればよい。

2 つずつ比較するため、ラウンド数は長さ - 1 になる。たとえば 3 つの数字なら、2 つずつ比較するには 2 ラウンドで足りる。

- アルゴリズム例

バブルソートは二重ループで実装する。

外側のループはソートのラウンド数を制御する。通常は配列長 - 1 にする。最後のループでは配列要素が 1 つだけ残り、比較の必要がないためである。

- 内側のループは主に、配列内の隣接要素の大小を比較し、位置を交換するかどうかを決める。

アルゴリズム実装。

#### 直接選択ソート

バブルソートより少し速い。

- 基本思想

指定したソート位置と他の配列要素をそれぞれ比較し、条件を満たせば要素値を交換する。

バブルソートとの違い:

隣接要素を交換するのではなく、指定したソート位置の要素を交換する。

- アルゴリズム例

各ラウンドで最小（または最大）の要素を 1 つ選び、最後に置く。

- アルゴリズム実装

#### 反転ソート

配列を逆順にするだけ。

- 基本思想

最後の要素と最初の要素を交換し、最後から 2 番目と 2 番目を交換する、という形で進める。

- アルゴリズム実装

public class Java6_5 {
    public static void main(String[] args) {
        int[] arr = {4, 7, 2, 6, 2, 6, 1, 4};
        //bubbleSort(arr);
        Java6_5 sorter = new Java6_5();
        // sorter.bubbleSort(arr);
        // sorter.selectSort(arr);
        sorter.reverseSort(arr);
        sorter.showArr(arr);
    }

    // 这里给出了两套方法
    // 一套是被注释掉的使用静态函数然后main函数中直接调用
    // 另一套是创建类的实例之后访问成员函数
    // 如果不需要访问实例变量 如辅助函数 工具函数 则静态方法比较好
    // 如果需要需要访问对象状态和成员变量则使用创建实例的方法
    // public static void bubbleSort(int[] arr){
    public void bubbleSort(int[] arr){
        // 外层控制排序轮数
        for (int i=0; i < arr.length - 1; i++){
            // 内层对比数组中每个邻近元素的大小
            for (int j=0; j < arr.length-(i+1); j++){
                // 比较相邻元素大小 若左大于右则交换位置
                if (arr[j]>arr[j+1]){
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
        /*
        这部分太常用了 拿出来写成一个函数
        for (int i : arr){
            System.out.print(i + " ");
        }
        */
    }

    public void selectSort(int[] arr) {
        int index;
        // 外层控制排序轮数
        for (int i = 1; i < arr.length; i++){
            // 每轮开始之前都要重置一下位置
            index = 0;
            // 因为index初始=0 故从arr[1]开始比较
            for (int j=1; j <= arr.length-i; j++){
                // 当前最大值的索引
                if (arr[j] > arr[index]){
                    index = j;
                }
            }
            // 交换最大值和最后一个未排序的元素的位置
            int temp = arr[arr.length-i];
            arr[arr.length-i] = arr[index];
            arr[index] = temp;
        }
    }

    public void reverseSort(int[] arr){
        int temp;
        for (int i=0; i<arr.length / 2; i++){
            temp = arr[i];
            arr[i] = arr[arr.length-(i+1)];
            arr[arr.length-(i+1)] = temp;
        }
    }

    public void showArr(int[] arr){
        for (int i : arr){
            System.out.print(i + " ");
        }
    }
}
