### Unfamiliar points

#### Inverse functions

The domain and range are swapped with those of the original function, and the graph is symmetric with the original function about `y=x`.

A strictly monotonic function must have an inverse function.

#### Four properties of functions

- Boundedness: if there exists `M` on interval `I` such that `|f(x)|≤M`, then `f(x)` is bounded on `I`.

![](/assets/posts/cnblogs-15159366/image-03.png)

- Monotonicity

![](/assets/posts/cnblogs-15159366/image-04.png)

- Parity: suppose the domain `D` of `f(x)` is symmetric about the origin, meaning if `x∈D`, then `-x∈D`. If for every `-x∈D`, `f(-x)=f(x)`, then `f(x)` is an even function. If for every `-x∈D`, `f(-x)=-f(x)`, then `f(x)` is an odd function.

*My own understanding: for even functions, the negative sign is eliminated; for odd functions, the negative sign can be factored out.*

## Function limits and continuity

![](/assets/posts/cnblogs-15159366/image-19.png)

Continuity and discontinuity are applications of function limits.

### Limits of functions

#### Definition [1]

![](/assets/posts/cnblogs-15159366/image-20.png)

- Tending to `∞`: after a certain point, all function values up to positive infinity are controlled within a certain interval. This resembles algorithmic convergence.

- Tending to `x0`: within a certain neighborhood, `x0±δ`, the function values are controlled within another interval, `A±ε`, where `A` is the limit as `x` tends to `x0`.

#### Properties

- Uniqueness

- The limit is a constant

- Local boundedness

- Local sign preservation

- Local inequality preservation: if `f(x)≤g(x)` in an interval, then `lim f(x)≤lim g(x)` in that interval.

- Squeeze theorem: if `f(x)≤h(x)≤g(x)` in an interval and `lim f(x)=lim g(x)=A`, then `lim h(x)=A`.

- Left limit equals right limit. The limit of a function at a point is unrelated to the function value at that point, for example at a discontinuity of a piecewise function.

### Continuity of functions

#### Definition

- Point of continuity: suppose function `f(x)` is defined in a neighborhood of `x0`, and `lim f(x)=f(0)`. Then `f(x)` is continuous at `x0`.

- Discontinuity

Removable discontinuity

- Jump discontinuity

- Infinite discontinuity

- Oscillatory discontinuity

Example [[2]](https://www.cnblogs.com/wakarimasu#fn2)

![](/assets/posts/cnblogs-15159366/image-25.png)

Removable and jump discontinuities are first-kind discontinuities, where both left and right limits exist.

Oscillatory and infinite discontinuities are second-kind discontinuities.
