        else {

        int n = 5;
        /*
        Idea:
        A diamond is split into upper and lower halves.
        The number of rows equals the number of columns in the middle row,
        and is usually odd.
        Set n = 5.
        The upper half ends at row n/2.
        As the row number increases:
        the number of stars in the middle row equals n,
        so the first row has n/2 spaces, decreasing by 1 each row.
        The number of stars in the first row is 1, then 3, 5, and so on.
        That is current row * 2 - 1.
        If i is the row counter, the decreasing space count should include i-1.

        The lower half can be handled by reversing the row counter i.
        It starts from n/2 + 1 and decreases by 1 each time.
         */
        // Print upper half
        for (int i = 1; i <= n / 2; i++) {
            for (int j = 1; j <= n / 2 - i + 1; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }

        // Print lower half
        for (int i = n / 2 + 1 ; i >= 1; i--) {
            for (int j = 1; j <= n / 2 - i + 1; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }

        int num = 1;

        Two while loops are a bit redundant.
        The idea is to use one variable to record the factorial denominator,
        and another variable to record the computed fraction.
        Since the denominator only multiplies one more number each time,
        it can be written in a single while loop.
        while (num <= 20){
            int k = 1;
            double tmp = 1;
            while (k <= num){
                tmp = tmp * k;
                k++;
            }
            sum = sum + (1.0 / tmp);

## Chapter 5: Strings

### 5.1 String class

#### Declaring strings

String variables must be initialized before use. This feels a bit obvious.

#### Creating strings

In Java, strings are managed as objects. Therefore, strings can be created like objects of other classes.

Common constructors:

- Create from character array `a`.

`String(char a[])`

- Extract part of character array `a` to create a string object.

`offset` indicates the starting position for extracting the string.

- `length` indicates the number of characters to extract.

`String(char a[], int offset, int length)`

Allocate a new `String` object representing the result of concatenating all elements in the character-array argument:

`String(char[] value)`

Note that `String(char a[])` and `String(char[] value)` are essentially not very different in code.

Usually the latter form is used.

But besides the constructors above, direct assignment is still the most common approach:

`String s = "string";`

```
public class Java5_1 {
    public static void main(String[] args){
        // Constructor
        // 1.String(char a[])
        char a[] = {'g', 'o', 'o', 'd'};
        String s = new String(a);
        // Equivalent to:
        String s1 = new String("good");

        // 2.String(char a[], int offset, int length)
        char b[] = {'g', 'o', 'o', 'd'};
        String s2 = new String(b, 2, 1);
        // Equivalent to:
        String s3 = new String("o");

        // 3.String(char[] value)
        String s5 = new String("good");

        // At this point s6 and s7 both reference the same string constant,
        // with the same entity and the same memory address.
        String s6, s7;
        s6 = "string";
        s7 = "string";

    }
}
```

### 5.2 Concatenating strings

#### Concatenating multiple strings

In Java, a string cannot be split across two lines directly. If it needs to be split, use `+`.

#### Concatenating other data types

After concatenation, everything becomes a string, but this should be used carefully.

```
public class Java5_2 {
    public static void main(String[] args){
        int booktime = 4;
        float practice = 2.5f;
        System.out.println("我每天花费" + booktime +
                "小时看书" + practice + "小时练习");
        // Parentheses have higher priority, so the two time values are added first.
        System.out.println("我每天花费" + booktime +
                "小时看书" + (practice + booktime) + "小时练习");
    }
}
```
