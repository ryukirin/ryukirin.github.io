## Chapter 5: Strings

### 5.3 Getting string information

#### Length information

`str.length();`

#### String lookup

Index range: `(0, length-1)`.

- `indexOf()`

Returns the index of the first occurrence. If not found, returns `-1`.

- `lastIndexOf()`

Returns the index of the last occurrence. If not found, returns `-1`.

If searching for `""`, it returns the length.

#### Get the character at a specified index

`str.charAt(int index)`

```
public class Java5_3 {
    public static void main(String[] args){
        String s = "hello world!";
        int len = s.length();
        System.out.println(len);

        // indexOf()
        int idx1;
        idx1 = s.indexOf('l');
        System.out.println(idx1);

        // lastIndexOf()
        int idx2, idx3;
        idx2 = s.lastIndexOf('l');
        idx3 = s.lastIndexOf("");
        System.out.println(idx2);
        System.out.println(idx3);
        System.out.println(s.length());

        // Get the character at a specified index.
        System.out.println(s.charAt(4));
    }
}
```

### 5.4 String operations

#### Get a substring

Substring extraction.

- `substring(int beingIndex)`

Extracts from the starting index to the end.

- `substring(int beingIndex, int endIndex)`

Specifies both start and end.

#### Remove spaces

`trim()`

Only removes spaces at the front and back.

#### String replacement

`str.replace(char oldChar, char newChar)`

If the target appears multiple times, all occurrences are replaced.

#### Check the beginning and ending of a string

- `startsWith()`: checks whether the string starts with a given string.

- `endsWith()`: checks whether the string ends with a given string.

#### Check whether strings are equal

Because `==` compares whether two variables have the same address.

Even if two strings have the same content, their addresses may differ, so the result can be `false`.

Therefore, use `equals()` and `equalsIgnoreCase()`.

The latter ignores case.

#### Compare two strings in lexicographical order

Based on Unicode values. If the string is before the compared string, the result is negative; if after, positive; if equal, 0.

`str.compareTo()`

#### Letter case conversion

Digits and non-letter characters are not affected.

- `toLowerCase()`: convert to lowercase.

- `toUpperCase()`: convert to uppercase.

#### String splitting

- `spilt(String sign)`: split according to the given character.

- `spilt(String sign, int limit)`: split according to the given character with a limit count.

```
public class Java5_4 {
    public static void main(String[] args){
        String s = " maybe this is an example ";

        // Get substring
        String s1 = s.substring(2);
        System.out.println(s1);

        String s2 = s.substring(2, 6);
        System.out.println(s2);

        // Remove spaces
        String s3 = s.trim();
        System.out.println(s3);

        // String replacement
        String s4 = s.replace("a", "b");
        System.out.println(s4);

        // Check beginning and ending
        System.out.println(s.startsWith(" "));
        System.out.println(s.endsWith("z"));

        // Check equality
        String s5 = " maybe this is an example ";
        String s6 = " Maybe this is an EXAMPLE ";
        System.out.println(s.equals(s5));
        System.out.println(s.equalsIgnoreCase(s6));

        // Compare strings lexicographically
        System.out.println(s.compareTo(s6));
        System.out.println(s.compareTo(s5));

        // Case conversion
        System.out.println(s.toUpperCase());

        // Split string
        String[] ss = s.split(" ");
        String[] ss1 = s.split(" ", 3);
        // Traverse array
        for (String i:ss) {
            System.out.println(i);
        }
        for (String i:ss1) {
            System.out.println(i);
        }
    }
}
```
