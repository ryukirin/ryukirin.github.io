# Watermelon Book | Datawhale October (4)

## Neural networks

### Neuron model

The M-P neuron model is a mathematical model used to simulate biological behavior.

In this model, a neuron receives input signals from `n` other neurons. These input signals are transmitted through weighted connections. The total input received by the neuron is compared with the neuron's threshold, and then processed by an activation function, which simulates inhibition or activation, to produce the neuron's output.

![](/assets/posts/cnblogs-15473679/image-01.png)

The ideal activation function is the step function, which maps input values to 0 or 1. However, the step function is discontinuous and not smooth, so in practice the Sigmoid function is used as the activation function.

Connecting many such neurons in a layered structure gives us a neural network.

We can view a neural network as a model formed by nesting expressions of the following form:

\[y=f\left(\sum_{i=1}^{n} w_{i} x_{i}-\theta\right)=f\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b\right)
\]

Single M-P neuron: perceptron, using `sgn`, a step function, as the activation function; or logistic regression, using `sigmoid` as the activation function.

Multiple M-P neurons: neural network.

### Perceptrons and multilayer networks

A perceptron consists of two layers of neurons. The input layer receives external input signals and passes them to the output layer. The output layer is an M-P neuron, also called a threshold logic unit.

![](/assets/posts/cnblogs-15473679/image-02.png)

For \(y_{j}=f(\sum_{i}w_{i}x_{i}-\theta_{j})=f(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta)\), if `f` is a step function:

- AND, \(x_1 \and x_2\): let \(w_1=w_2=1,\theta=2\), then \(y=f(1\cdot x_1+1\cdot x_2-2)\). Only when \(x_1=x_2=1\), `y = 1`.

- OR, \(x_1 \or x_2\): let \(w_1=w_2=1,\theta=0.5\), then \(y=f(1\cdot x_1+1\cdot x_2-0.5)\). When \(x_1=1\) or \(x_2=1\), `y = 1`.

- NOT, \(\neg x_1\): let \(w_1=-0.6,w_2=0,\theta=-0.5\), then \(y=f(-0.6\cdot x_1+0\cdot x_2+0.5)\). When \(x_1=1\), `y=0`; when \(x_1=0\), `y=1`.

More generally, given a training data set, weights \(w_i(i=1,2,...,n)\) and threshold \(\theta\) can be learned. The threshold \(\theta\) can be viewed as the connection weight \(w_{n+1}\) corresponding to a dummy node whose fixed input is `-1.0`. This unifies threshold learning and weight learning.

For a training sample \((x,y)\), if the current perceptron output is \(\hat{y}\), the perceptron adjusts the weights as:

\[\begin{aligned}
w_{i} & \leftarrow w_{i}+\Delta w_{i} \\
\Delta w_{i} &=\eta(y-\hat{y}) x_{i}
\end{aligned}
\]

where \(\eta \in (0,1)\) is called the learning rate.

A more detailed version is introduced below.

From a geometric perspective, given a linearly separable data set \(T\), the learning goal of a perceptron is to find a hyperplane that perfectly separates positive and negative samples in \(T\). Here \(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta\) is the hyperplane equation.

The hyperplane in an \(n\)-dimensional space, \(\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b=0\right. , where  \left.\boldsymbol{w}, \boldsymbol{x} \in \mathbb{R}^{n}\right)\):

- The hyperplane equation is not unique.

- The normal vector \(\boldsymbol{w}\) is perpendicular to the hyperplane.

- The normal vector \(\boldsymbol{w}\) and displacement term \(b\), or \(-\theta\), determine a unique hyperplane.

- The half-space pointed to by the normal vector \(\boldsymbol{w}\) is the positive space, and the other half is the negative space.

#### Perceptron learning strategy

Randomly initialize \(\boldsymbol{w}, b\), substitute all training samples into the model, and find misclassified samples. Suppose the set of misclassified samples is \(M \subseteq T\). For any misclassified sample \((\boldsymbol{x}, y) \in M\), when \(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta \geqslant 0\), the model output is \(\hat{y}=1\), while the true label is `y = 0`. Conversely, when \(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta<0\), the model output is \(\hat{y}=0\), while the true label is `y = 1`. Combining the two cases, the following formula always holds:

\[(\hat{y}-y)\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta\right) \geq 0
\]

Therefore, given data set \(T\), its loss function can be defined as:

\[L(\boldsymbol{w}, \theta)=\sum_{\boldsymbol{x} \in M}(\hat{y}-y)\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}-\theta\right)
\]

This loss function is clearly non-negative. If there are no misclassified points, the loss is 0. Also, the fewer misclassified points there are, and the closer they are to the hyperplane, the smaller the loss.

#### How to optimize the loss function

Given data set:

\[T=\left\{\left(\boldsymbol{x}_{1}, y_{1}\right),\left(\boldsymbol{x}_{2}, y_{2}\right), \ldots,\left(\boldsymbol{x}_{N}, y_{N}\right)\right\}
\]

where \(\boldsymbol{x}_{i} \in \mathbb{R}^{n}, y_{i} \in\{0,1\}\), solve parameters \(\boldsymbol{w}, \theta\) so that they minimize the loss function:

\[\min _{\boldsymbol{w}, \theta} L(\boldsymbol{w}, \theta)=\min _{\boldsymbol{w}, \theta} \sum_{\boldsymbol{x}_{i} \in M}\left(\hat{y}_{i}-y_{i}\right)\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}-\theta\right)
\]

where \(M \subseteq T\) is the set of misclassified samples. If the threshold \(\theta\) is viewed as a dummy node with fixed input `-1`:

\[-\theta=-1 \cdot w_{n+1}=x_{n+1} \cdot w_{n+1}
\]

then the minimization problem can be further simplified as:

\[\min _{\boldsymbol{w}} L(\boldsymbol{w})=\min _{\boldsymbol{w}} \sum_{\boldsymbol{x}_{\boldsymbol{i}} \in M}\left(\hat{y}_{i}-y_{i}\right) \boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}
\]

#### Perceptron learning algorithm

When the misclassified sample set \(M\) is fixed, the gradient of the loss function \(L(\boldsymbol{w})\) is:

\[\nabla_{\boldsymbol{w}} L(\boldsymbol{w})=\sum_{\boldsymbol{x}_{i} \in M}\left(\hat{y}_{i}-y_{i}\right) \boldsymbol{x}_{\boldsymbol{i}}
\]

The perceptron learning algorithm uses stochastic gradient descent. That is, during minimization, it does not update using the gradient of all misclassified points in \(M\) at once; instead, it randomly selects one misclassified point at a time for the gradient step. The update rule for weight \(\boldsymbol{w}\) is:

\[\begin{array}{c}
\boldsymbol{w} \leftarrow \boldsymbol{w}+\Delta \boldsymbol{w} \\
\Delta \boldsymbol{w}=-\eta\left(\hat{y}_{i}-y_{i}\right) \boldsymbol{x}_{i}=\eta\left(y_{i}-\hat{y}_{i}\right) \boldsymbol{x}_{i}
\end{array}
\]

Correspondingly, the update rule for one component \(w_i\) of \(\boldsymbol{w}\) is:

\[\Delta w_{i} =\eta(y-\hat{y}) x_{i}
\]

The final solved \(\boldsymbol{w}\) is usually not unique.

The perceptron only applies the activation function in the output-layer neuron, so it has only one functional neuron layer. Its learning ability is very limited and can only solve linearly separable problems.

To solve non-linearly separable problems such as XOR, multilayer functional neurons are needed.

### Multilayer feedforward networks

The following image shows a more common neural network. Neurons in adjacent layers are fully connected. There are no same-layer connections and no cross-layer connections. This is usually called a multilayer feedforward neural network.

![](/assets/posts/cnblogs-15473679/image-04.png)

The input layer only receives external input and does not perform functional processing. The hidden layer and output layer contain functional neurons and process signals. Therefore, Figure 5.6(a) is a two-layer neural network. To avoid ambiguity, it can be called a single-hidden-layer neural network. The learning process of a neural network is the process of adjusting connection weights between neurons and thresholds of each functional neuron.

![](/assets/posts/cnblogs-15473679/image-05.png)

Treat the neural network, denoted as `NN`, as a feature-processing function:

\[\boldsymbol{x} \in \mathbb{R}^{d} \rightarrow \operatorname{NN}(\boldsymbol{x}) \rightarrow \boldsymbol{y}=\boldsymbol{x}^{*} \in \mathbb{R}^{l}
\]

For single-output regression, attach a neuron mapping \(\mathbb{R}^{l} \rightarrow \mathbb{R}\), for example a neuron without an activation function:

\[y=\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}^{*}+b
\]

For classification, attach a neuron mapping \(\mathbb{R}^{l} \rightarrow[0,1]\), for example a neuron using the \(\operatorname{sigmoid}\) activation function. The loss function can be cross entropy:

\[y=\frac{1}{1+e^{-\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}^{*}+b\right)}}
\]

During model training, the neural network automatically learns and extracts useful features. In this sense, machine learning takes another step toward fully automated data analysis.

#### Backpropagation

Suppose all activation functions in the multilayer feedforward network are \(\operatorname{sigmoid}\), and the task is a multi-output regression task. The loss function can therefore use mean squared error; for classification tasks, cross entropy is used. For a training sample \(\left(\boldsymbol{x}_{k}, \boldsymbol{y}_{k}\right)\), where \(\boldsymbol{y}_{k}=\left(y_{1}^{k}, y_{2}^{k}, \ldots, y_{l}^{k}\right)\), suppose the output of the multilayer feedforward network is \(\hat{\boldsymbol{y}}_{k}=   \left(\hat{y}_{1}^{k}, \hat{y}_{2}^{k}, \ldots, \hat{y}_{l}^{k}\right)\). The mean squared error, or loss, for this single sample is:

\[E_{k}=\frac{1}{2} \sum_{j=1}^{l}\left(\hat{y}_{j}^{k}-y_{j}^{k}\right)^{2}
\]

The error backpropagation algorithm, or BP algorithm, is a parameter-update algorithm based on stochastic gradient descent:

\[\begin{array}{l}
w \leftarrow w+\Delta w \\
\Delta w=-\eta \nabla_{w} E
\end{array}
\]

So we only need to derive \(\nabla_w E\), the first-order partial derivative, or gradient, of loss function \(E\) with respect to parameter \(w\), using the chain rule. It is worth mentioning that \(\mathrm{NN}(\boldsymbol{x})\) is usually an extremely complex non-convex function, and does not have the nice mathematical properties of convex functions. Therefore, stochastic gradient descent cannot guarantee convergence to a global minimum. In most cases, it reaches a local minimum.

Formula derivation reference: [Pumpkin Book](https://datawhalechina.github.io/pumpkin-book/#/chapter5/chapter5?id=_510)

BP algorithm steps:

```
'''
Input: training set D; learning rate lr
Process:
1: Randomly initialize all connection weights and thresholds in the network within (0, 1)
2: repeat
3:    for all (x_k, y_k) ∈ D do
4:      Compute current sample output y_hat_k using current parameters and Equation 1;
5:      Compute the output-layer neuron gradient term g_j using Equation 8;
6:      Compute the hidden-layer neuron gradient term e_h using Equation 13;
7:      Update connection weights w_hj, v_ih and thresholds theta_j, gamma_h using Equations 9-12
8:    end for
9: until the stopping condition is reached
Output: multilayer feedforward neural network determined by connection weights and thresholds
'''
```

### Deep learning

Typical deep learning is a very deep neural network. However, multilayer hidden neural networks are difficult to train directly with classic algorithms such as BP, because when errors propagate through many hidden layers, they often diverge and fail to converge to a stable state.

Unsupervised layer-wise training is an effective method for training networks with many hidden layers. The main idea is to train one hidden layer at a time. During training, the output of the previous hidden layer is used as input, and the output of the current hidden layer becomes the input to the next hidden layer. This is called pretraining. After all pretraining is completed, the entire network is fine-tuned.

Another strategy for reducing training cost is weight sharing, meaning a group of neurons uses the same connection weights. This strategy is used in CNNs.

We can also understand deep learning as feature learning or representation learning: through multilayer processing, the initial low-level feature representation is gradually transformed into high-level feature representation, so that complex classification and other learning tasks can be completed with a simple model.
