## Chapter 4: Flow control

### 4.1 Compound statements

A compound statement creates a scope for local variables.

Local variables defined inside it can only be used within that compound statement.

```
public class Java4_1 {
    public static void main(String[] args){
        int x = 20;
        {// This block is a compound statement.
            int y = 40;
            System.out.println(y);
            int z = 245;
            boolean b;
            {// This block is also a compound statement.
                b = y > z;
                System.out.println(b);
            }
        }
        // System.out.println(y, z, b);
        // y, z, and b cannot be used here.
        String word = "hello java";
        System.out.println(word);
    }
}
```

### 4.2 Conditional statements

#### if statement

- If there is only one statement after the `if` condition, `{}` can be omitted, but it is better not to omit them.

- If there is no statement after the `if` condition, `{}` can be omitted and `;` can be added, or just use `{}`.

#### if...else statement

It can be simplified into a ternary operation. See the code for details.

#### if...else if statement and switch multi-branch statement

See the code for details.

```
public class Java4_2 {
    public static void main(String[] args){
        boolean b = false;
        if (b){}
        if (b);

        int x = 45;
        int y = 12;
        if (x > y){
            System.out.println("变量x大于变量y");
        }
        if (x < y){
            System.out.println("变量x小于变量y");
        }

        int math = 95;
        int english = 56;
        if (math > 60){
            System.out.println("数学及格了");
        } else {
            System.out.println("数学没及格");
        }
        if (english > 60){
            System.out.println("英语及格了");
        } else {
            System.out.println("英语没及格");
        }

        int m;
        int n = -8;
        if (n > 0){
            m = n;
        }
        else {
            m = -n;
        }
        System.out.println(m);
        // Can be simplified as:
        m = n > 0?n:-n;
        System.out.println(m);

        if (math >= 60 && math < 90){
            System.out.println("数学:及格");
        }
        else if (math >= 90) {
            System.out.println("数学:优秀");
        }
        else {
            System.out.println("数学:不及格");
        }

        String user_input = "1";
        switch (user_input){
            // The expression after case must be an integer, character, or string.
            case "1":
                System.out.println("1.取钱");
                // Without break, the next case statement is executed until break is encountered.
                break;
            case "2":
                System.out.println("2.存钱");
                break;
            case "end":
                break;
            // default is optional.
            default:
                System.out.println("请输入1或者2，退出输入end");
        }
    }
}
```

### 4.3 Loop statements

#### while loop

See the code for details.

#### do...while loop

Execute the loop body first, then evaluate `while`.

#### for loop

`for(expression1; expression2; expression3){}`

- Expression 1: initialize variables.

- Expression 2: loop condition.

- Expression 3: variable update operation.

#### foreach statement

Mainly convenient for traversing arrays.

```
for(element variable x : traversed object obj){
    Java statement using x;
}
```

```
public class Java4_3 {
    public static void main(String[] args){
        int i = -2;
        while (i <= -1){
            i++;
            System.out.println("run while");
        }

        do {
            i++;
            System.out.println("run do while "+ i +" times.");
        }while (i <= -3);

        // If the initial value is 0 and you want n loops, write variable < n.
        // If the initial value is 1 and you want n loops, write variable <= n.
        for (int j=1; j<=3; j++){
            System.out.println("run for "+ j +" times.");
        }

        int arr[] = {7, 10, 1};
        System.out.println("item in arr is: ");
        for (int x : arr){
            System.out.println(x);
        }
    }
}
```

### 4.5 Loop control

#### break

Exit the current loop body, but only one loop level.

To exit the outer loop:

```
labelName: loopBody {
    break labelName;
}
```

#### continue statement

Skip the remaining statements in this iteration and start the next iteration.

Labels are also supported.

```
labelName: loopBody {
    continue labelName;
}
```

```
public class Java4_4 {
    public static void main(String[] args){
        System.out.println("---------break---------");
        for (int i=1; i<=4; i++){
            System.out.println("running for1 in "+ i +" times");
            for (int j=1; j<=4; j++){
                System.out.println("running for2 in "+ j +" times");
                if (j >= 1){
                    // Exits for2 but continues for1.
                    break;
                }
            }
        }

        System.out.println("-------跳出外层循环-------");
        // Exit the outer loop.
        Loop: for (int i=1; i<=4; i++){
            System.out.println("running Loop in "+ i +" times");
            for (int j=1; j<=4; j++){
                System.out.println("running for3 in "+ j +" times");
                if (j >= 1){
                    // Exits for1, labeled Loop.
                    break Loop;
                }
            }
        }

        System.out.println("--------continue--------");
        for (int i=1; i<=4; i++){
            System.out.println("running for in "+ i +" times");
            if (i > 1){
                // Skips the rest of this iteration and continues the loop.
                continue;
            }
            System.out.println("running after if in the "+ i +"time");
        }

        System.out.println("-------跳出外层循环-------");
        // Continue the outer loop.
        Loop: for (int i=1; i<=2; i++){
            System.out.println("running Loop in "+ i +" times");
            for (int j=1; j<=4; j++){
                System.out.println("running for in "+ j +" times");
                if (j > 1){
                    // Exits the inner for loop and directly starts the next Loop iteration.
                    continue Loop;
                }
                System.out.println("running after if in the "+ i +"time");
            }
        }
    }
}
```
