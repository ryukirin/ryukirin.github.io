I tried a lot of approaches before this, such as deleting the desktop shortcut and recreating it from the Steam library, but none of them worked.

I even uninstalled and reinstalled the game.

I also checked the folder where Steam stores game icons, but the icon for the game I wanted to fix simply was not there.

Eventually I found that the most direct fix was to change the shortcut icon manually.

I will use *Octopath Traveler* as the example here. That was the broken shortcut on my desktop, and it has now been fixed.

### Right-click the desktop shortcut -> Properties -> Web Document -> Change Icon

![](/assets/posts/cnblogs-14529720/image-01.png)

According to the tutorials I had seen before, clicking Browse and opening `your-install-path\Steam\steam\games` should show the `.ico` files for the games. In my case, it did not.

So I directly located the folder that contains the `.exe` file for *Octopath Traveler*.

### Steam Library -> Octopath Traveler -> Settings -> Manage -> Browse local files

![](/assets/posts/cnblogs-14529720/image-02.png)

The file should usually be under:

```
your-install-path\Steam\steamapps\common\game-name
```

For *Octopath Traveler*, I found the executable directly in the game root directory. For *Don't Starve*, I found it under the `bin` directory.

![](/assets/posts/cnblogs-14529720/image-03.png)

![](/assets/posts/cnblogs-14529720/image-04.png)

### Right-click the `.exe` file to create a shortcut, then right-click the generated shortcut -> Properties -> Shortcut -> Change Icon

![](/assets/posts/cnblogs-14529720/image-05.png)

### Copy that path into the Change Icon dialog for the desktop shortcut.

### Done.
