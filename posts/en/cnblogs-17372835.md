# Regularization

## Lecture Translation

### Modeling Nonlinear Relations

- We studied linear models in the previous lecture, but nonlinear models are also important.

- Consider a model defined as a linear combination of basis functions.

> In mathematics, a basis function is an element of a particular basis in a function space. Every continuous function in a function space can be represented as a linear combination of basis functions, just as every vector in a vector space can be represented as a linear combination of basis vectors.

\[f(\boldsymbol{x}) = \beta_1\phi_1(\boldsymbol{x})+\beta_2\phi_2(\boldsymbol{x})+\cdots+\beta_p\phi_p(\boldsymbol{x})
\]

where \(\phi_i(\boldsymbol{x})\) is a predefined basis function.

- If \(\phi_i(\boldsymbol{x})\) is a nonlinear mapping, then \(f(\boldsymbol{x})\) becomes a nonlinear function with respect to \(\boldsymbol{x}\).

#### Example

Polynomial model: \(
\phi_{1}(x)=x, \phi_{2}(x)=x^{2}, \phi_{3}(x)=x^{3}\)

\[f(x)=\beta_{1} x+\beta_{2} x^{2}+\beta_{3} x^{3}
\]

This is a cubic function of \(x\).

### Basis Function Regression

- On the other hand, \(f(\boldsymbol{x})\) is still a linear function with respect to \(\beta\).

\[f(x)=\begin{bmatrix}
 \phi_1(\boldsymbol{x})&\cdots&\phi_p(\boldsymbol{x})
\end{bmatrix}
\begin{bmatrix}
 \beta_1\\
 \vdots \\
\beta_p
\end{bmatrix}
\]

Here, \(\begin{bmatrix}
 \phi_1(\boldsymbol{x})&\cdots&\phi_p(\boldsymbol{x})
\end{bmatrix}\) is the feature vector, so LSE can be applied directly to estimate \(\beta\).

- Define

\[\boldsymbol{\phi} = \begin{bmatrix}
\phi_1(x_1) & \phi_2(x_1)  & \cdots  & \phi_p(x_1)\\
 \vdots & \vdots &  &\vdots \\
\phi_1(x_n) & \phi_2(x_n)  & \cdots  & \phi_p(x_n)\\
\end{bmatrix}
\]

Then the LSE for \(\beta\) can be written as

\[\hat{\beta} = (\boldsymbol{\phi}^{\top}\boldsymbol{\phi})^{-1}\boldsymbol{\phi}^{\top}\boldsymbol{y}
\]

### Gaussian Basis Function

- Gaussian radial basis functions are commonly used.

\[\phi_i(\boldsymbol{x})=\exp(-\frac{\left \| \boldsymbol{x}-\boldsymbol{c}_i \right \|^2 }{2\vartheta ^2} )
\]

Here, \(\boldsymbol{c}_i\) is a predefined center parameter. $\left | \boldsymbol{x}-\boldsymbol{c}_i \right |^2 $ is the Euclidean distance, or L2 norm, between an arbitrary point \(\boldsymbol{x}\) and a center point \(\boldsymbol{c}_i\). \(\vartheta\) is the width parameter.

![](/assets/posts/cnblogs-17372835/image-01.png)

- Example with six basis functions:

\[\begin{aligned}
\phi_{1}(\boldsymbol{x})= & \exp \left(-\|\boldsymbol{x}-0\|^{2}\right) \\
\phi_{2}(\boldsymbol{\boldsymbol{x}})= & \exp \left(-\|\boldsymbol{\boldsymbol{x}}-1\|^{2}\right) \\
& \vdots \\
\phi_{6}(\boldsymbol{x})= & \exp \left(-\|\boldsymbol{x}-5\|^{2}\right)
\end{aligned}
\]

### Ridge Regression

For details, see [Machine Learning Algorithms in Practice: Ridge Regression and LASSO](https://zhuanlan.zhihu.com/p/30535220).

- Introduce a penalty mechanism on \(\beta\):

\[\min_{\beta}\left \| \boldsymbol{y}-\boldsymbol{X}\beta \right \|^2+\lambda \left \| \beta \right \| ^2
\]

When \(\lambda>0\), it is the regularization coefficient.

From here on, use \(\boldsymbol{X}\) instead of \(\boldsymbol{\phi}\).

- The penalty on \(\beta\) suppresses the flexibility of the resulting model.

- When \(\lambda=0\), ridge regression reduces to LSE.

By taking the derivative:

\[\hat{\beta}_{ridge} = (\boldsymbol{X}^{\top}\boldsymbol{X}+\lambda\boldsymbol{I})^{-1}\boldsymbol{X}^{\top}\boldsymbol{y}
\]

#### Example

- Same setting as the overfitting example.

![](/assets/posts/cnblogs-17372835/image-04.png)

- In practice, \(λ\) should be selected appropriately, for example by cross-validation.

### Regularization

- In general, introducing a penalty mechanism on parameters is called regularization, and it is widely used across many machine learning algorithms.

- The effectiveness of regularization can be explained theoretically through the tradeoff between estimation bias and variance.

### Bias of Ridge Regression

- Assume \(\boldsymbol{y}=\boldsymbol{X} \boldsymbol{\beta}+\varepsilon\), where \(\beta\) is the unknown true parameter, \(\varepsilon \sim \mathcal{N}\left(\mathbf{0}, \sigma^{2} \boldsymbol{I}\right)\), and \(\boldsymbol{X}^{\top}\boldsymbol{X}\) is invertible.

- Ridge regression is a biased estimator. See the appendix for the proof.

\[\begin{aligned}
\mathbb{E}\left[\hat{\boldsymbol{\beta}}_{\text {ridge }}\right]
& = \mathbb{E}\left[\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{X}^{\top} \boldsymbol{y}\right]\\
& = \beta-\lambda(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I})^{-1}\beta
\end{aligned}
\]

where \(\lambda(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I})^{-1}\beta\) is the bias.

- Two extreme cases:

If \(λ=0\), the bias becomes 0, because ridge regression is then equivalent to LSE.

- If \(λ→  ∞\), the bias is \(β\), because with an extremely strong penalty, the learned parameter is always 0.

### Variance of Ridge Regression

- The variance of the ridge estimator is

\[\begin{aligned}
\mathbb{V}\left[\hat{\boldsymbol{\beta}}_{\text {ridge }}\right]
& = \mathbb{V}\left[\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{X}^{\top} \boldsymbol{y}\right]\\
& = \vartheta ^2\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{X}^{\top} \boldsymbol{X}\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1}
\end{aligned}
\]

The proof is almost the same as for LSE.

- The larger \(\lambda\) is, the smaller the variance becomes.

- This shows that ridge regression can reduce estimation variance while giving up unbiasedness.

### Analyzing Prediction Error

- Consider the error for a given test point \(x_{te}\), compared with the true value:

\[E(\hat{\boldsymbol{\beta}})=\left(\boldsymbol{x}_{\mathrm{te}}^{\top} \boldsymbol{\beta}-\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}
\]

- Since \(\hat{\beta}\) depends on \(y\), and as we saw in the previous lecture \(y\) is a random variable, we consider its expectation:

\[\mathbb{E}[E(\hat{\boldsymbol{\beta}})]=\mathbb{E}\left[\left(\boldsymbol{x}_{\mathrm{te}}^{\top} \boldsymbol{\beta}-\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right]
\]

### Bias-Variance Decomposition

- \(\mathbb{E}[E(\hat{\boldsymbol{\beta}})]\) can be decomposed as follows. See the appendix for the proof.

\[\begin{aligned}
\mathbb{E}[E(\hat{\boldsymbol{\beta}})] & =\mathbb{E}\left[\left(\boldsymbol{x}_{\text {te }}^{\top} \boldsymbol{\beta}-\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right] \\
& =\left(\boldsymbol{x}_{\text {te }}^{\top}(\boldsymbol{\beta}-\mathbb{E}[\hat{\boldsymbol{\beta}}])\right)^{2}+\boldsymbol{x}_{\text {te }}^{\top} \mathbb{V}[\hat{\boldsymbol{\beta}}] \boldsymbol{x}_{\text {te }}
\end{aligned}
\]

Here, \((\boldsymbol{\beta}-\mathbb{E}[\hat{\boldsymbol{\beta}}])\) is the bias, and \(\mathbb{V}[\hat{\boldsymbol{\beta}}]\) is the variance.

- Therefore, the bias-variance tradeoff is about balancing the first and second terms.

#### Regularization

Regularization tries to reduce expected prediction error by reducing variance, at the cost of introducing bias.

- In general, when a model has higher flexibility, its estimation variance tends to be larger.

## Appendix

### Bias of Ridge Regression

\[\begin{array}{l}
\mathbb{E}\left[\hat{\boldsymbol{\beta}}_{\text {ridge }}\right]=\mathbb{E}\left[\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{X}^{\top} \boldsymbol{y}\right] \\
=\mathbb{E}\left[\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{X}^{\top}(\boldsymbol{X} \boldsymbol{\beta}+\varepsilon)\right] \\
=\left(\boldsymbol{U} \boldsymbol{\Lambda} \boldsymbol{U}^{\top}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{U} \boldsymbol{\Lambda} \boldsymbol{U}^{\top} \boldsymbol{\beta} \text { \# Eigendecomposition } \boldsymbol{X}^{\top} \boldsymbol{X}=\boldsymbol{U} \boldsymbol{\Lambda} \boldsymbol{U}^{\top} \\
=\boldsymbol{U}\left(\boldsymbol{I}+\lambda \boldsymbol{\Lambda}^{-1}\right)^{-1} \boldsymbol{U}^{\top} \boldsymbol{\beta} \\
=\boldsymbol{U}\left(\boldsymbol{I}-\boldsymbol{I}+\left(\boldsymbol{I}+\lambda \boldsymbol{\Lambda}^{-1}\right)^{-1}\right) \boldsymbol{U}^{\top} \boldsymbol{\beta} \\
=\boldsymbol{\beta}-\boldsymbol{U}\left(\boldsymbol{I}-\left(\boldsymbol{I}+\lambda \boldsymbol{\Lambda}^{-1}\right)^{-1}\right) \boldsymbol{U}^{\top} \boldsymbol{\beta} \\
=\boldsymbol{\beta}-\boldsymbol{U}\left(\boldsymbol{I}+\lambda \boldsymbol{\Lambda}^{-1}\right)^{-1}\left(\left(\boldsymbol{I}+\lambda \boldsymbol{\Lambda}^{-1}\right)-\boldsymbol{I}\right) \boldsymbol{U}^{\top} \boldsymbol{\beta} \\
=\boldsymbol{\beta}-\lambda \boldsymbol{U}(\boldsymbol{\Lambda}+\lambda \boldsymbol{I})^{-1} \boldsymbol{U}^{\top} \boldsymbol{\beta} \\
=\boldsymbol{\beta}-\lambda\left(\boldsymbol{X}^{\top} \boldsymbol{X}+\lambda \boldsymbol{I}\right)^{-1} \boldsymbol{\beta}
\end{array}
\]

### Eigendecomposition

Decompose the eigenvalues of \(\boldsymbol{X}^{\top} \boldsymbol{X}\):

\[\boldsymbol{X}^{\top} \boldsymbol{X}=\boldsymbol{U} \boldsymbol{\Lambda} \boldsymbol{U}^{\top}
\]

where \(\boldsymbol{U}\) is an orthogonal matrix, \(\left(\boldsymbol{U}^{\top} \boldsymbol{U}=\boldsymbol{U} \boldsymbol{U}^{\top}=\boldsymbol{I}\right)\), and \(\boldsymbol{\Lambda}\) is a diagonal matrix whose diagonal entries are eigenvalues.

Then

\[\begin{aligned}
\left(\boldsymbol{U} \boldsymbol{\Lambda} \boldsymbol{U}^{\top}+\lambda \boldsymbol{I}\right)^{-1} & =\left(\boldsymbol{U}(\boldsymbol{\Lambda}+\lambda \boldsymbol{I}) \boldsymbol{U}^{\top}\right)^{-1} \\
& =\boldsymbol{U}(\boldsymbol{U}(\boldsymbol{\Lambda}+\lambda \boldsymbol{I}))^{-1} \\
& =\boldsymbol{U}(\boldsymbol{\Lambda}+\lambda \boldsymbol{I})^{-1} \boldsymbol{U}^{\top}
\end{aligned}
\]

Note that for nonsingular matrices \(\boldsymbol{A}\) and \(\boldsymbol{B}\), \((\boldsymbol{A} B)^{-1}=\boldsymbol{A}^{-1} \boldsymbol{B}^{-1}\).

### Bias-Variance Decomposition

\[\begin{array}{l}
\mathbb{E}\left[\left(\boldsymbol{x}_{\mathrm{te}}^{\top} \boldsymbol{\beta}-\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right] \\
=\mathbb{E}\left[\left(\boldsymbol{x}_{\text {te }}^{\top} \boldsymbol{\beta}-\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]+\mathbb{E}\left[\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right]-\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right] \\
=\mathbb{E}\left[\left(\boldsymbol{x}_{\text {te }}^{\top} \boldsymbol{\beta}-\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]\right)^{2}+2\left(\boldsymbol{x}_{\text {te }}^{\top} \boldsymbol{\beta}-\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]\right)\left(\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]-\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right)\right. \\
\left.+\left(\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]-\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right] \\
=\mathbb{E}\left[\left(\boldsymbol{x}_{\text {te }}^{\top} \boldsymbol{\beta}-\mathbb{E}\left[\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right]\right)^{2}\right] \\
+2\left(\boldsymbol{x}_{\mathrm{te}}^{\top} \boldsymbol{\beta}-\mathbb{E}\left[\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right]\right) \underbrace{\mathbb{E}\left[\left(\mathbb{E}\left[\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right]-\boldsymbol{x}_{\mathrm{te}}^{\top} \hat{\boldsymbol{\beta}}\right)\right]}_{=0} \\
\left.+\mathbb{E}\left[\left(\mathbb{E}\left[\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right]-\boldsymbol{x}_{\text {te }}^{\top} \hat{\boldsymbol{\beta}}\right)^{2}\right]\right] \\
=\left(\boldsymbol{x}_{\text {te }}^{\top}(\underbrace{\boldsymbol{\beta}-\mathbb{E}[\hat{\boldsymbol{\beta}}]}_{\text {bias }})\right)^{2}+\boldsymbol{x}_{\text {te }}^{\top} \underbrace{\mathbb{E}\left[(\hat{\boldsymbol{\beta}}-\mathbb{E}[\hat{\boldsymbol{\beta}}])(\hat{\boldsymbol{\beta}}-\mathbb{E}[\hat{\boldsymbol{\beta}}])^{\top}\right]}_{\text {variance }} \boldsymbol{x}_{\mathrm{te}} \\
\end{array}
\]

\[\mathbb{E}\left[\hat{\boldsymbol{\beta}}_{\text {ridge }}\right]=(1-\frac{1}{20})\beta=\frac{19}{20}\beta

\[\mathbb{E}\left[\hat{\boldsymbol{\beta}}_{\text {ridge }}\right]=(1-\frac{11}{30})\beta=\frac{19}{30}\beta
