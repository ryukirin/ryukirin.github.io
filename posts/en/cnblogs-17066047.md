# Operating Neo4j with py2neo

## Installation

### Java 17

#### Download

If Java is not installed, install Java first. Refer to the official manual for the required version.

[https://neo4j.com/docs/operations-manual/current/installation/windows/](https://neo4j.com/docs/operations-manual/current/installation/windows/)

Download link:

[https://www.oracle.com/java/technologies/downloads/#java17](https://www.oracle.com/java/technologies/downloads/#java17)

#### Install

After downloading, click the executable file and install step by step.

After installation, configure environment variables.

On Windows:

- In user variables: create a `JAVA_HOME` variable, with the value set to the installation directory.

Example: `D:\Java\jdk-17`

- In environment variables: add `%JAVA_HOME%\bin` to `Path`.

#### Test

Press `Win+R`, open `cmd`, and enter `java -version`. If the Java version shows 17, it is OK.

### Neo4j

Neo4j Community Edition:

[https://neo4j.com/download-center/#community](https://neo4j.com/download-center/#community)

Follow the instructions after downloading.

Then press `Win+R`, open `cmd`, and run:

```
installation-directory\bin\neo4j console
```

For example:

```
D:\neo4j\bin\neo4j console
```

Visit `http://localhost:7474/` to see the operation page.

## Python operations

### Drivers

#### neo4j-driver

It was originally named `neo4j-driver`, but is now changed to `neo4j`.

```
pip install neo4j
```

#### py2neo

This post mainly uses this package.

```
pip install py2neo
```

### Basic example

```
from py2neo import Node, Relationship, Graph

USERNAME = username
PASSWORD = password

# Create a graph connection.
graph = Graph('http://localhost:7474', auth=(USERNAME, PASSWORD), name='neo4j')

# Create nodes.
# Node(labels, *properties)
a = Node('label', name='a')
b = Node('label', name='b')

# Draw nodes in the graph.
graph.create(a)
graph.create(b)

# Create and draw a relationship.
# Extra properties can be added, or omitted.
# Relationship(start_node, type, end_node, **properties)
r1 = Relationship(a, 'to', b, name='to')
graph.create(r1)
```

### Database operations, mainly nodes and relationships

Before all operations, prepare:

```
USERNAME = your username
PASSWORD = your password
NAME = your database name
graph = Graph('http://localhost:7474', auth=(USERNAME, PASSWORD), name=NAME)
```

#### Create

- Add a node

```
node = Node("Person", name="Alice", id=node_1)
graph.create(node)
```

The first argument is the node label, the second is the node name shown in the Neo4j graph, and the third is the node ID, which I define as a custom primary key for easier lookup.

None of them are strictly required. The latter two are node properties.

- Add a label

```
node.add_label("Women")
```

- Add properties

```
node["age"] = 25
```

Set the value of the `new_name` property to `value`.

- Add a relationship

```
node_1 = Node("Person", name="Alice", id="node_1")
node_2 = Node("Person", name="Bob", id="node_2")

# 1. First method
relationship = Relationship(node_1, "KNOWS", node_2)
graph.create(relationship)

# 2. Second method
class Knows(Relationship): pass
ab = Knows(node_1, node_2)
graph.create(ab)

# 3. Third method
KNOWS = Relationship.type("KNOWS")
ab = KNOWS(node_1, node_2)
graph.create(ab)
```

- Add a subgraph

```
node_1 = node_matcher.match("Person").where(id='node_1').first()
node_2 = node_matcher.match("Person").where(id='node_2').first()
node_3 = node_matcher.match("Person").where(id='node_3').first()
node_4 = node_matcher.match("Person").where(id='node_4').first()

KNOWS = Relationship.type("KNOWS")
ab = KNOWS(node_1, node_2)
ac = KNOWS(node_1, node_3)
s = ab | ac
graph.create(s)
```

#### Read

Before querying:

```
from py2neo import NodeMatcher, RelationshipMatcher
node_matcher = NodeMatcher(graph)
relation_matcher = RelationshipMatcher(graph)
```

- Query nodes

```
node = node_matcher.match("Person").where(name="Alice").first()
```

The argument of `match()` is the label and is optional. The argument of `where()` is the properties and is also optional.

The result of `match` is a list, so `.first()` needs to be added.

- Query relationships

Query all relationships in the graph:

```
relation = list(relation_matcher.match((node, None), r_type='idea'))
```

If you want to query by node or relationship name, the prerequisite is that the node has already been found, or that you know the relationship type.

```
# Only the start node is known.
relation = list(relation_matcher.match((start_node, None)))

# Only the end node is known.
relation = list(relation_matcher.match((None, end_node)))

# Only the relationship type is known.
relation = list(relation_matcher.match(r_type='Knows'))

# Other queries can be built by combining the three patterns above.

# The obtained relation can be traversed to get start_node, end_node, and properties.
relation[0].start_node['id']
```

- Query labels

```
# Check whether node has a label.
node.has_label(label)

# Query node labels.
node.labels
```

- Query properties

```
# Query the value of the property named name.
# 1. First method
node["name"]
# 2. Second method
node_1.get("name")

# Return the number of properties.
len(node)

# Return a dictionary of properties.
dict(node)
# Return node properties and values as a list of tuples.
node.items()
# Return property names.
node.key()
# Return property values.
node.values()
```

#### Update

The prerequisite for updating is that the target object has already been found.

- Modify a node

```
# Modify a node property.
node['name'] = 'May'

# setdefault(): if the property exists, return its value; otherwise set it to 'May'.
node.setdefault('name', default = 'May')　

# Up to this point, only local data has been modified.
# To make it effective in Neo4j, this line must be added.
graph.push(node)
```

- Modify a relationship

```
# Modify a relationship property.
relation['name'] = 'has'

# setdefault(): if the property exists, return its value; otherwise set it to 'May'.
relation.setdefault('name', default = 'has')　

# Up to this point, only local data has been modified.
# To make it effective in Neo4j, this line must be added.
graph.push(relation)
```

#### Delete

To delete a node, first delete the relationships on that node.

- Delete a relationship

```
graph.separate(relationship)
```

- Delete a node

```
graph.delete(node)
```

- Clear the database

```
graph.delete_all()
```

- Delete labels

```
# Delete a label if it exists.
node.remove_label('Person')

# Delete all labels.
node.clear_label()
```

- Delete properties

```
# Delete one property. If it does not exist, a KeyError is raised.
del node["name"]

# Delete all properties.
node.clear()
```
