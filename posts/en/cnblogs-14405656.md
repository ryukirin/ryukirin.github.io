## Update on 2021-03-13

When I configured the new computer, I still ran into a few issues.

I tried installing Jupyter Notebook in my PyTorch environment through Anaconda, but the installation kept failing.

After checking a few references, I eventually solved it with:

```
conda install nb_conda
```

That said, this method also failed the first time I tried it. It only worked after I ran the command once in the `base` environment and once in the `pytorch` environment.
