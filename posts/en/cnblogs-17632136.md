## Chapter 6: Arrays

### 6.1 Array overview

An array is a collection of data with the same data type.

### 6.2 Creating and using one-dimensional arrays

Essentially, this is a linear collection of data of the same type.

#### Creating one-dimensional arrays

Array indexes start from 0.

- Declare first, then allocate memory using the `new` operator.

`elementType arrayName[]`

- `elementType[] arrayName`

After declaration, the array cannot be accessed immediately. Memory still needs to be allocated, and the length must be specified.

`arrayName = new elementType[numberOfElements]`

Allocate memory while declaring:

- `elementType arrayName[] = new elementType[numberOfElements]`

- `elementType[] arrayName = new elementType[numberOfElements]`

#### Initializing one-dimensional arrays

- Allocating memory for an array is also a kind of initialization. All elements default to 0.

- Initialize inside `new`, for example `int arr[] = new int[]{1, 2, 3, 4};`

- Omit `new` and initialize directly, for example `int arr[] = {1, 2, 3, 4};`

Once array length is defined, it cannot be changed.

#### Using one-dimensional arrays

Traverse with a `for` loop.

```
public class Java6_1_2 {
    public static void main(String[] args) {
        // Create one-dimensional arrays.
        // Declare first, then allocate memory with new.
        int arr[];
        int[] arr1;
        arr = new int[3];
        arr1 = new int[4];
        // Allocate memory while declaring.
        int arr2[] = new int[5];

        // Initialize one-dimensional arrays.
        int[] arr3 = new int[6];
        System.out.println(arr3[0]);
        int arr4[] = {1, 2, 3, 4};

        // Use one-dimensional arrays.
        for (int i = 0; i<arr4.length; i++){
            System.out.println(arr4[i]);
        }
        for (int i : arr4){
            System.out.println(i);
        }
    }
}
```

### 6.3 Creating and using two-dimensional arrays

#### Creating two-dimensional arrays

- Declare first, then allocate memory using the `new` operator.

`elementType arrayName[][]`

- `elementType[][] arrayName`

For high-dimensional arrays, memory can be allocated:

Allocate memory directly for each dimension, for example `a = new int[2][4]`.

- Allocate memory separately for each dimension, for example `a = new int[2][]; a[0] = new int[2]; a[1] = new int[3];`

Allocate memory while declaring.

#### Initializing two-dimensional arrays

Example: `int arr[][] = {{1, 2, 3}, {4, 5, 6}}`

#### Using two-dimensional arrays

```
public class Java6_3 {
    public static void main(String[] args) {
        int arr[][] = {{1, 2, 3}, {4, 5}};
        for (int i=0; i<arr.length; i++){
            for (int j=0; j<arr[i].length; j++){
                System.out.println(arr[i][j]);
            }
        }
        for (int[] i : arr){
            for (int j : i){
                System.out.println(j);
            }
        }
    }
}
```

### 6.4 Basic array operations

The `Arrays` class is used:

`import java.util.Arrays;`

#### Traversing arrays

The previous two sections already wrote two methods:

- `for`

- `foreach`

#### Filling or replacing array elements

- `Arrays.fill(int[] a, int value)`

`a`: the array whose elements are replaced.

- `value`: the value stored into the array.

`Arrays.fill(int[] a, int fromIndex, int toIndex, int value)`

- `a`: the array whose elements are replaced.

- `fromIndex`: index of the first element to fill with the specified value, inclusive.

- `toIndex`: index of the last element to fill with the specified value, exclusive.

- `value`: the value stored into the array.

#### Sorting arrays

`Arrays.sort()` sorts in ascending order.

For `String`, sorting is lexicographical ascending order.

#### Copying arrays

- `Arrays.copyOf(arr, int newLength)`

- `Arrays.copyOfRange(arr, int formIndex, int toIndex)`

#### Searching arrays

`Arrays.binarySearch` uses binary search.

The array must be sorted before using it.

- `Arrays.binarySearch(Object[] a, Object key)`

`a`: the array to search.

- `key`: the value to search for.

- Return value: index.

`Arrays.binarySearch(Object[] a, int formIndex, int toIndex, Object key)`

```
import java.util.Arrays;
public class Java6_4 {
    public static void main(String[] args) {
        // Fill or replace array elements.
        int arr[] = {1, 2, 3, 4, 5};
        Arrays.fill(arr, 5);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");
        Arrays.fill(arr, 2, 3, 1);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");

        // Sort array.
        Arrays.sort(arr);
        for (int i : arr) {
            System.out.print(i);
        }
        System.out.println("");

        // Copy array.
        int newArr[] = Arrays.copyOf(arr, 10);
        for (int i : newArr) {
            System.out.print(i);
        }
        System.out.println("");

        int newArr1[] = Arrays.copyOfRange(arr, 0, 3);
        for (int i : newArr1) {
            System.out.print(i);
        }
        System.out.println("");

        // Search array.
        System.out.println("1的索引为" + Arrays.binarySearch(arr, 1));
        // The result is 2, because binarySearch returns the first found index.
        // It uses binary search, so the first position checked is the middle.
        System.out.println("1的索引为" + Arrays.binarySearch(arr, 5));
    }
}
```
