# How to Create a Custom Memory Class

This post mainly defines a custom Memory class used in LangChain.

Original article: [How to create a custom Memory class](https://python.langchain.com/docs/modules/memory/how_to/custom_memory)

## Translation

Although LangChain already provides several predefined memory types, it is still quite possible that you will want to add your own memory type for a specific application. This note introduces how to do that.

In this notebook, we add a custom memory type to `ConversationChain`. To add a custom memory class, we need to import the base memory class and subclass it.

```
from langchain import OpenAI, ConversationChain
from langchain.schema import BaseMemory
from pydantic import BaseModel
from typing import List, Dict, Any
```

In this example, we write a custom memory class that uses spaCy to extract entities and stores entity information in a simple hash table. During conversation, we inspect the input text, extract all entities, and place any known information about them into the context.

- Note that this implementation is very simple and fragile, and may not be useful in production. Its purpose is to demonstrate that custom memory can be added.

For this, we need spaCy.

### A Brief Introduction to spaCy

spaCy is an NLP development library that is easy to get started with. In simple terms, the spaCy usage here first loads a language model, then runs a sentence through it. Several NLP tasks are completed at the same time, including tokenization and part-of-speech tagging, and the results are stored in a `Doc` object.

Quick-start tutorial: [Advanced NLP with spaCy](https://course.spacy.io/zh)

### Custom Memory Example

```
import spacy

nlp = spacy.load("en_core_web_lg")
```

```
class SpacyEntityMemory(BaseMemory, BaseModel):
    """Memory class used to store entity information."""

    # Dictionary used to store entity information.
    entities: dict = {}
    # Key used to select entity information added to the prompt.
    memory_key: str = "entities"

    def clear(self):
        self.entities = {}

    @property
    def memory_variables(self) -> List[str]:
        """Define variables added to the prompt."""
        return [self.memory_key]

    def load_memory_variables(self, inputs: Dict[str, Any]) -> Dict[str, str]:
        """Load memory variables. In this example, that means entities."""
        # Get the input text and run it through spaCy.
        doc = nlp(inputs[list(inputs.keys())[0]])
        # Extract known information for entities if it exists.
        entities = [
            self.entities[str(ent)] for ent in doc.ents if str(ent) in self.entities
        ]
        # Return the aggregated entity information for insertion into the context.
        return {self.memory_key: "\n".join(entities)}

    def save_context(self, inputs: Dict[str, Any], outputs: Dict[str, str]) -> None:
        """Save the current conversation context into the buffer."""
        # Get the input text and run it through spaCy.
        text = inputs[list(inputs.keys())[0]]
        doc = nlp(text)
        # For each mentioned entity, save the information into the dictionary.
        for ent in doc.ents:
            ent_str = str(ent)
            if ent_str in self.entities:
                self.entities[ent_str] += f"\n{text}"
            else:
                self.entities[ent_str] = text
```

Now we define a prompt that receives entity information and user input.

```
from langchain.prompts.prompt import PromptTemplate

template = """The following is a friendly conversation between a human and an AI. The AI is talkative and provides lots of specific details from its context. If the AI does not know the answer to a question, it truthfully says it does not know. You are provided with information about entities the Human mentions, if relevant.

Relevant entity information:
{entities}

Conversation:
Human: {input}
AI:"""
prompt = PromptTemplate(input_variables=["entities", "input"], template=template)
```

Then we put everything together.

```
llm = OpenAI(temperature=0)
conversation = ConversationChain(
    llm=llm, prompt=prompt, verbose=True, memory=SpacyEntityMemory()
)
```

In the first example, because there is no prior knowledge about Harrison, the `Relevant entity information` field is empty.

## Personal Test Example

I wanted a custom memory that can extract one specific key-value from the JSON generated by GPT-3.5 each time.

In this example, the JSON format is:

```
{
  "current_step": current step name;
  "result_info":
	{
    	"step": generate one JSON object in the step list for each default step
    	[{
      		"name": step name generated from the default steps;
      		"ok": whether confirmation with the user is complete, yes/no;
		}, ...];
    	"is_over": whether all confirmations with the user are complete, yes/no;
    	"next_step": if "is_over" is "yes", write the next step name; if "is_over" is "no", write "";
	};
  "reply": the facilitator's utterance, must contain content;
}
```

The key I want to extract is `"reply"`, so I created a new `RecentKConversationMemory` class.

*Note that the `RecentKConversationMemory` class here is not general-purpose. It is only useful for this specific task.*

```
from langchain.schema import BaseMemory
from typing import List, Dict, Any


class RecentKConversationMemory(BaseMemory):
    """Memory class that stores a specific key value from JSON files in the most recent five conversation turns."""
    # Define the list used to store recent conversations. Default is empty.
    recent_conversations: list = []
    # Define the key used to select information added to the prompt. Default is "recent_conversations".
    memory_key: str = "recent_conversations"
    # Define prefixes. Default values are "Human" and "AI".
    human_prefix: str = "Human"
    ai_prefix: str = "AI"
    # Define how many turns to keep. Default is 5.
    k: int = 5

    def clear(self):
        self.recent_conversations = []

    @property
    def memory_variables(self) -> List[str]:
        """Define variables added to the prompt."""
        return [self.memory_key]

    def load_memory_variables(self, inputs: Dict[str, Any]) -> Dict[str, str]:
        """Load memory variables. Here this is the content of a specific key."""
        recent_json_value = ""

        # Traverse the most recent k conversation turns.
        for conversation in self.recent_conversations[-self.k:]:
            # Check whether content with the key "reply" exists.
            if "reply" in eval(conversation["output"]["response"]).keys():
                human = f"{self.human_prefix}: " + conversation["input"]['input']
                ai = f"{self.ai_prefix}: " + eval(conversation["output"]["response"])["reply"]
                recent_json_value += "\n" + "\n".join([human, ai])

        return {self.memory_key: recent_json_value}

    def save_context(self, inputs: Dict[str, Any], outputs: Dict[str, Any]) -> None:
        """Save the current conversation context into the buffer."""
        # Store the input and output of each turn as one conversation.
        conversation = {
            "input": inputs,
            "output": outputs
        }
		# If there are more than five, pop the earliest conversation.
        self.recent_conversations.append(conversation)
        if len(self.recent_conversations) > self.k:
            self.recent_conversations.pop(0)
```
