# Watermelon Book | Datawhale October (5)

## Support vector machines

From a geometric perspective, for linearly separable models, a support vector machine finds the hyperplane that is farthest from both positive and negative samples.

Compared with the perceptron, its solution is unique and unbiased, and it has better generalization performance.

### Hyperplane

An \(n\)-dimensional hyperplane \(\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b=0\right. , where  \boldsymbol{w}, \boldsymbol{x} \in \mathbb{R}^{n}  )\):

- The hyperplane equation is not unique.

- The normal vector \(\boldsymbol{w}\) and displacement term \(b\) determine a unique hyperplane.

- The normal vector \(\boldsymbol{w}\) is perpendicular to the hyperplane. When scaling \(\boldsymbol{w}, b\), a negative scaling factor changes the direction of the normal vector.

- The half-space pointed to by the normal vector \(\boldsymbol{w}\) is the positive space, and the other half is the negative space.

- The distance from any point \(\boldsymbol{x}\) to the hyperplane is:

\[r=\frac{\left|\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b\right|}{\|\boldsymbol{w}\|}
\]

### Geometric margin

For a given data set \(X\) and hyperplane \(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b=0\), define the geometric margin of any sample point \(\left(\boldsymbol{x}_{i}, y_{i}\right), y_i \in \{-1,1\}, i=1,2,\ldots,m\) in data set \(X\) with respect to the hyperplane as:

\[\gamma_{i}=\frac{y_{i}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}+b\right)}{\|\boldsymbol{w}\|}
\]

When the classification is correct, \(\gamma_i>0\), and the geometric margin is equivalent to the distance from the point to the hyperplane. When the classification is incorrect, \(\gamma_i<0\).

For a given data set \(X\) and hyperplane \(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b=0\), define the geometric margin of data set \(X\) with respect to the hyperplane as the minimum geometric margin among all sample points in \(X\):

\[\gamma=\min _{i=1,2, \ldots, m} \gamma_{i}
\]

### Support vector machine

We want to find the hyperplane whose geometric margin on the data set is maximized, and then apply a `sign` function for classification.

\[y=\operatorname{sign}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b\right)=\left\{\begin{aligned}
1, & \boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b>0 \\
-1, & \boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}+b<0
\end{aligned}\right.
\]

So its essence is the same as the perceptron: it is still solving for a hyperplane.

The hyperplane with the largest geometric margin must be the hyperplane farthest from both positive and negative samples, for two reasons:

- When the hyperplane does not correctly separate positive and negative samples, the smallest geometric margin corresponds to a misclassified point, so \(\gamma<0\).

- When the hyperplane correctly separates the samples, \(\gamma \geqslant 0\), and the closer the hyperplane is to the center, the larger \(\gamma\) is.

Strategy: given linearly separable data set \(X\), suppose the sample in \(X\) with the smallest geometric margin is \(\left(\boldsymbol{x}_{\min }, y_{\min }\right)\). Then the process of finding the hyperplane for a support vector machine can be converted into the following constrained optimization problem:

\[\begin{array}{ll}
\max \quad \gamma \\
\text { s.t. } \quad \gamma_{i} \geqslant \gamma, \quad i=1,2, \ldots, m \\
\end{array}
\]

\[\begin{array}{ll}
\max _{\boldsymbol{w}, b} \frac{y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right)}{\|\boldsymbol{w}\|} \\
\text { s.t. } \quad \frac{y_{i}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}+b\right)}{\|\boldsymbol{w}\|} \geqslant \frac{y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right)}{\|\boldsymbol{w}\|},  \quad i=1,2, \ldots, m \\
\end{array}
\]

\[\begin{array}{ll}
\max _{\boldsymbol{w}, b} \frac{y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right)}{\|\boldsymbol{w}\|} \\
\text { s.t. } \quad y_{i}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}+b\right) \geqslant y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right), \quad i=1,2, \ldots, m
\end{array}
\]

Suppose the optimal solution is \(\left(\boldsymbol{w}^{*}, b^{*}\right)\). Then \(\left(\alpha \boldsymbol{w}^{*}, \alpha b^{*}\right), \alpha \in \mathbb{R}^{+}\) is also an optimal solution, and the hyperplane does not change. Therefore, some restriction must be placed on \(\boldsymbol{w}, b\) so that the optimization problem has a solvable unique solution. Let \(y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right)=1\), because for a specific \(\left(\boldsymbol{x}_{\min }, y_{\min }\right)\), there is exactly one \(\alpha\) that makes \(y_{\min }\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{\min }+b\right)=1\). The optimization problem further becomes:

\[\begin{array}{ll}
\max _{\boldsymbol{w}, b} & \frac{1}{\|\boldsymbol{w}\|} \\
\text { s.t. } & y_{i}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}+b\right) \geqslant 1, \quad i=1,2, \ldots, m
\end{array}
\]

For easier subsequent computation, apply an equivalent transformation, adding a square and `1/2` for convenience:

\[\begin{array}{ll}
\min _{\boldsymbol{w}, b} & \frac{1}{2}\|\boldsymbol{w}\|^{2} \\
\text { s.t. } & 1-y_{i}\left(\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_{i}+b\right) \leqslant 0, \quad i=1,2, \ldots, m
\end{array}
\]

This is an optimization problem with inequality constraints, and it is a convex optimization problem. It can therefore be solved directly by many specialized methods for convex optimization. For support vector machines, Lagrange duality is commonly used.
