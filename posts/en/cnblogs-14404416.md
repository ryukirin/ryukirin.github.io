One extra note: when setting up PyTorch, if your GPU is usable, you can first check which CUDA versions your graphics card supports, and then choose the matching PyTorch installation command.

To check the GPU: open Task Manager -> Performance -> GPU. If the GPU is shown in the upper-right corner, the driver is installed correctly.
