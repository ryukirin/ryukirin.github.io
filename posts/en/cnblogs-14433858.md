## File creation

### Create an empty file

```
os.mknod("test.txt")
```

**This method does not work on Windows.**

### Open a file directly, and create it if it does not exist

```
import os

file = open("test01.txt",'w')
file.close()
with open("test01.txt", 'w', encoding="utf-8") as f:
    f.write("test")
with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
test
```

```
# Wrapped file-creation helper
def text_create(name):
    # File save location. By default, this is the project root.
    savepath = ""
    filename = savepath + name + '.txt'
    filename = name + '.txt'
    file = open(filename, 'w')
    file.close()

text_create('test')
```

## File writing

**By "writing" here, I mean more than just the `write` operation.**

### write

The `write` operation clears all existing file content and then writes the new content.

```
with open("test.txt", 'w', encoding="utf-8") as f:
    f.write("the 1st line\n")
# Read operation
with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
print("-------------------")
with open("test.txt", 'w', encoding="utf-8") as f:
    f.write("the 2nd line\n")

with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
the 1st line

-------------------
the 2nd line
```

### append

The append operation adds content after the existing content.

```
with open("test.txt", 'a', encoding="utf-8") as f:
    f.write("the 3rd line\n")
# Read operation
with open("test.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
```

### Delete content

```
with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
    
# f.truncate(n) truncates from the beginning after the nth character and clears the rest.
# In other words, it keeps the first n characters.
with open("test01.txt", 'a', encoding="utf-8") as f:
    f.truncate(2)

with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
    print("-------------")
    
# Clear file content
with open("test01.txt", 'w', encoding="utf-8") as f:
    f.truncate()

with open("test01.txt", 'r', encoding="utf-8") as f:
    print(f.read())
```

```
test
te
-------------
```

```
# Delete file
os.remove("test01.txt")

# Check whether the file exists
os.path.exists("test01.txt")
```

```
False
```

## File reading

### open() + close()

```
# Open file
f = open('test.txt','r',encoding = 'utf-8')

# Then use read() to view the file content.
print(f.read())

# Close file
f.close()
```

```
the 2nd line
the 3rd line
```

**Notes**

- If the file encoding is not specified, and the file encoding differs from the current default encoding, reading or writing the file may fail. In Python 3, the encoding can be specified through the `encoding` parameter.

- Always call `close()` at the end to close the file.

- If an error occurs while reading or writing the file, the file may fail to close properly even if `close()` is used.

### with open()

`with` is equivalent to calling `close()` automatically. The file is closed automatically, which is safer and also avoids forgetting to close files manually.

```
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
```

```
with open('test.txt','a',encoding = 'utf-8') as f:
    f.write("test\ntest2\ntest3\n")
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
```

```
the 2nd line
the 3rd line
test
test2
test3
```

### Several ways to read

```
# Read everything
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.read())
    print(f.tell()) # Print pointer position. After read(), the pointer is at the end.
    f.seek(0) # Move pointer: 0 = file head/default, 1 = current position, 2 = file end.
    print(f.tell())
    print("-----------")

# Read the first line
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readline())
    print("-----------")
    
# Read the first n characters. Here n = 2.
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readline(2))
    print("-----------")
    
# Read all content at once and return a list by line.
with open('test.txt','r',encoding = 'utf-8') as f:
    print(f.readlines())
    print("-----------")
```

-----------
