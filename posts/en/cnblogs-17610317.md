## Chapter 3: Java language basics

### 3.1 Java main class structure

- Package declaration

- Import API libraries, for example `import xxx;`

- Declare class

- Declare member variables and local variables

- Write the main method

```
// Package declaration
// This line can only be written when the file is in the Number package.
package Number;

// Must match the .class filename; in other words, this is where the filename is written.
public class Java3_1 {
    // Global variable, also called member variable. Keyword: static.
    // Global variables are declared inside the class body.
    static String s1 = "你好";
    // Main method
    // public static void main function, with a String array parameter
    public static void main(String[] args) {
        // Local variable, declared inside the method body.
        String s2 = "Java";
        System.out.println(s1);
        System.out.println(s2);
    }
}
```

### 3.2 Basic data types

- Numeric types

Integer types: `byte`, `short`, `int`, `long`

- Floating-point types: `float`, `double`

Character type

Boolean type

#### Integer types

Decimal, octal, and hexadecimal.

Notes:

- Except for 0, decimal numbers cannot start with 0.

- Octal numbers must start with 0.

- Hexadecimal numbers must start with `0X` or `0x`.

| Data type | Memory space, 8 bits = 1 byte | Value range |
| --- | --- | --- |
| byte | 8 bits | -128~127 |
| short | 16 bits | -32768~32767 |
| int | 32 bits | -2147483648~2147483647 |
| long | 64 bits | -9223372036854775808~19223372036854775807 |

#### Floating-point types

- Single-precision floating-point type: `float`

- Double-precision floating-point type: `double`

Note:

- By default, floating-point literals are `double`. To use `float`, add `F` or `f` after the number.

| Data type | Memory space, 8 bits = 1 byte | Value range |
| --- | --- | --- |
| float | 32 bits | 1.4E-45~3.4028235E38 |
| double | 64 bits | 4.9E-324~1.7976931348623157E308 |

#### Character type

- `char`

Stores only one character.

- Takes 16 bits of memory.

- Must be represented with single quotes when defined.

Escape characters start with `\`.

| Escape character | Meaning | Escape character | Meaning |
| --- | --- | --- | --- |
| \ddd | 1-3 octal digits | \ uxxxx | 4 hexadecimal digits |
| ' | single quote | \ | backslash |
| \t | tab | \r | carriage return |
| \n | newline | \b | backspace |
| \f | form feed |  |  |

#### Boolean type, logical type

Only has two values: `true` and `false`.

```
public class Java3_2 {
    public static void main(String[] args){
        System.out.println("----------------1.整数类型----------------");
        // Integer type
        // Define int variable x.

        // bug text: java: integer number too large

        // bug text: java: incompatible types: possible lossy conversion from double to float
        float f1 = 13.23f;
        double d = 638.678;

        System.out.println("单精度浮点类型：" + f1);

        System.out.println("例子：");
        char word = 'd', word2 = '@';
        int p = 23045, p2 = 45213;
        System.out.println("d在Unicode表中的顺序位置是：" + (int)word);
        System.out.println("d在Unicode表中的顺序位置是：" + (int)word2);
        System.out.println("Unicode表中的第23045位是：" + (char)p);
        System.out.println("Unicode表中的第45213位是：" + (char)p2);
        System.out.println("转义字符");
        char c3 = '\\';
        char c4 = '\u2605';
        char c5 = '\123';
        String str = "says \'OH, I see, thx.\'";
        String str1 = "table name1\ttable name2\t\ntable value1\ttable value2\t";
        char c6 = 'a';
        char c7 = '\b';
        char c8 = 'c';
        char c9 = '\f';
        System.out.println("反斜杠" + c3 + "星星" + c4);
        // Single quote
        System.out.println(str);
        // Tab and newline
        System.out.println(str1);
        /*

        System.out.print(c6);
        System.out.print(c7);
        System.out.println(c8);
        System.out.println(c9);
    }
```

### 3.3 Constants and variables

- Identifier

Composed of letters, underscores, dollar signs, and digits in any order. Case-sensitive.

- The first character cannot be a digit.

- It cannot be a reserved keyword such as `int`.

Keywords:

| int | public | this | finally | boolean | abstract |
| --- | --- | --- | --- | --- | --- |
| continue | float | long | short | throw | throws |
| return | break | for | static | new | interface |
| if | goto | default | byte | do | case |
| strictfp | package | super | void | try | switch |
| else | catch | implements | private | final | class |
| extends | volatile | while | synchronized | instanceof | char |
| protected | import | transient | double |  |  |

#### Variable declaration

Defining a variable tells the compiler the variable type so that memory space can be allocated.

If the value stored in that memory space can change, the memory space is called a variable. The variable name is the name of that memory space.

When declaring a variable, you may assign a value or leave it unassigned, but the variable type must be declared.

#### Constant declaration

A constant is usually also called a final variable.

It can only be assigned once in the whole program and requires the `final` keyword.

Constant names are usually written in all uppercase letters.

#### What happens when variables are declared

Memory:

- OS area

- Program area

- Data area

When the program executes:

1. Program code is loaded into the program area.
2. Space is allocated in the data area for the two variables.

#### Variable scope

- Member variables: defined in the class body.

Static variables can be used across classes through `ClassName.staticVariableName`.

- Instance variables

Local variables:

- Defined inside a class method body and valid only in the current code block.

- Their lifecycle depends on the method.

When the method is called, the Java virtual machine allocates memory for local variables in that method.

- After the method call ends, the memory occupied by local variables in the method is released, and the local variables are destroyed.

When a local variable and a member variable have the same name, the member variable temporarily becomes invalid inside that method.

```
public class Java3_3 {
    // When a constant is a member variable, it must be assigned when declared.
    static final double PI = 3.14;
    // This one does not need to be assigned during declaration.
    static int age = 18;
    // Instance variable
    int x = 45;
    // Static variable
    static int y = 90;
    static int times = 3;
    public static void main(String[] args){
        final int number;
        // number = 1235;
        // bug text: java: variable number might already have been assigned
        // number is a constant and can only be assigned once.
        age = 22;
        number = 1236;
        System.out.println("常量PI的值为："+PI);
        System.out.println("赋值后number的值为："+number);
        System.out.println("int型变量age的值为："+age);

        // When names conflict, the variable defined in this method takes priority.
        int times = 4;
        System.out.println("times的值为：" + times);
    }
}
```
