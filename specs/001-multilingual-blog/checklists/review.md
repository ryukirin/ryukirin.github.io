## 需求完整性

- [x] CHK001 是否明确规定了首页、博客列表、文章详情三类页面各自必须承载的核心内容？ [Completeness, Spec §用户故事 1-3]
- [x] CHK002 是否明确规定了语言切换入口固定出现在 `header`，而不是由实现阶段自行决定位置？ [Clarity, Spec §FR-002]
- [x] CHK003 是否完整定义了中文、英文、日文三种支持语言，以及超出支持范围时的默认行为？ [Completeness, Spec §FR-002a, Spec §假设]
- [x] CHK004 是否完整定义了个人简介、博客文章、本地化内容版本三类内容对象及其必要字段？ [Completeness, Spec §关键实体]
- [x] CHK005 是否明确写出博客图片必须来自仓库内本地文件，而不是把图像来源留给实现阶段猜测？ [Completeness, Spec §FR-009a]
- [x] CHK006 是否明确写出文章需要支持 LaTeX 行内与块级数学公式，而不是仅笼统写“支持技术文章”？ [Clarity, Spec §FR-009b]

## 需求清晰度

- [x] CHK007 “简短个人简介”是否被限制到足以支撑首页布局的篇幅，而不是停留在主观描述？ [Ambiguity, Spec §个人简介, Spec §边界情况]
- [x] CHK008 “简洁大方、显眼但不喧宾夺主”等视觉描述，是否已经转化为可执行的布局、层级或组件位置要求？ [Clarity, Spec §体验意图, Plan §宪章检查]
- [x] CHK009 “当前显示状态”是否明确区分为“目标语言正文”“中文原文回退”“暂不可用”三种可判断状态？ [Clarity, Spec §FR-005, Spec §SC-004]
- [x] CHK010 “按需补充英文和日文版本”是否明确说明为手动维护，而不是给人留下自动翻译仍可能在 v1 中存在的空间？ [Consistency, Spec §Clarifications, Spec §FR-004, Plan §约束条件]

## 需求一致性

- [x] CHK011 规格中的多语策略、计划中的技术约束、任务中的实现路径，是否都一致表述为“v1 不接自动翻译、译文手动维护”？ [Consistency, Spec §Clarifications, Plan §约束条件, Tasks §US3]
- [x] CHK012 规格中的文章目录结构与 frontmatter 规则，是否与计划、数据模型、契约文档中的结构保持一致，没有出现平铺目录或不同字段名？ [Consistency, Spec §FR-010, Plan §项目结构, Data Model §4-5, Contracts §内容契约]
- [x] CHK013 首页与博客页面是否使用同一套语言切换与回退术语，而不是在不同部分混用“译文”“语言版本”“本地化状态”等未定义称呼？ [Consistency, Spec §FR-005~FR-006a, Contracts §路由契约]

## 验收标准质量

- [x] CHK014 首访语言匹配、首页理解成本、首页到文章点击路径等成功标准，是否都可以被客观验证而不是主观判断？ [Measurability, Spec §SC-001, Spec §SC-001a, Spec §SC-002]
- [x] CHK015 对于“正文完整可读、不横向溢出”的要求，是否已经具体到可用于评审的页面范围和场景类型？ [Acceptance Criteria, Spec §SC-003]
- [x] CHK016 “中文原文不被阻塞”这一发布要求，是否已经明确到能与手动维护译文流程区分开？ [Clarity, Spec §SC-005, Spec §SC-006]

## 场景覆盖

- [x] CHK017 是否同时覆盖了首次访问、手动切换语言、目标语言缺失、原文入口返回这几类主要阅读路径？ [Coverage, Spec §用户故事 1-3]
- [x] CHK018 是否对“个人简介缺失译文”与“博客文章缺失译文”分别提出了要求，而不是只写了文章回退逻辑？ [Coverage, Spec §FR-006, Tasks §T015, Gap]
- [x] CHK019 是否说明了当文章包含图片、代码块、列表、引用和公式时，这些内容都必须参与多语言阅读体验定义？ [Coverage, Spec §FR-008~FR-009b]

## 边界情况覆盖

- [x] CHK020 是否定义了本地图像缺失、路径错误、尺寸过大时的需求，而不是默认实现会自然处理？ [Edge Case, Spec §边界情况]

- [x] CHK022 是否定义了浏览器语言不在支持列表内时的默认回退行为？ [Completeness, Spec §FR-002a]
